### 四、"更多"按钮修复

#### 问题
`display: none` 与 `transform: translateY(100%)` 同时使用导致动画失效

#### 解决
```css
/* 移除 display: none，改用 pointer-events */
.more-menu {
  transform: translateY(100%);
  pointer-events: none;  /* 隐藏时不可点击 */
}
.more-menu.open {
  transform: translateY(0);
  pointer-events: auto;  /* 显示时可点击 */
}
```
