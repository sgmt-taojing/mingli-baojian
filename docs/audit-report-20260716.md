## 三、修复记录

### 修复1：十神表TENGAN正偏反转

**问题：** 代码中同性=正、异性=偏，与古制《渊海子平》"同性为偏、异性为正"的规则完全相反。

**修复内容：**
- divination-core.js 第1420-1429行：TENGAN表全部10个日干的 财/才、官/杀、印/枭 映射互换
- annual-fortune.js 第70-81行：TENGAN_LOCAL 表同步修复

**验证结果：** 10日干×9关系=90组映射全部验证通过

### 修复2：大运步进×3→×1

**问题：** `computeDayun()` 函数中大运干支步进公式中多余的 `* 3` 导致每柱跳过3个干支位置。

**修复内容：**
```javascript
// 修复前
const zhiIdx = (startZhi + i * direction * 3 + 36) % 12;
const ganIdx = (startGan + i * direction * 3 + 40) % 10;
// 修复后
const zhiIdx = (startZhi + i * direction + 36) % 12;
const ganIdx = (startGan + i * direction + 40) % 10;
```

**验证结果：** 月柱甲子阳男顺排 → 甲子→乙丑→丙寅→丁卯 ✅

### 修复3：纳音命名统一

**问题：** group 8（戊寅己卯）的 `nayin` 字段写为"城墙土"，古制《三命通会》标准名称为"城头土"。

**修复内容：** 全文件90处"城墙土"统一替换为"城头土"

### 修复4：口诀错字

**问题：** koujue-daily.js 第2条口诀"春不容金，秋不容木"，原文应为"秋不容土"。

**修复内容：** `"秋不容木"` → `"秋不容土"`

### 修复5：系统性内容冗余清理

**问题：** authoritative-knowledge-base.js 中三种重复段落标记：
- 【理论深化】（八字命理） — 126处重复
- 【姓名学理论深化】 — 33处重复
- 【五行学说理论深化】 — 41处重复
- 【纳音五行精要】 — 90处（266处引用）

**清理策略：** 每种标记保留第一次完整版，删除所有后续重复段落
