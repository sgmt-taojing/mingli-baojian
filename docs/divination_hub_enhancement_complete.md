- ✅ 常用名用字400+ 
- ✅ 数据存储为JavaScript对象：`KANGXI_STROKES`

**三才五格计算逻辑** (完整实现):
```
天格 = 姓氏笔画 + 1
人格 = 姓氏笔画 + 名字第一字笔画
地格 = 名字各字笔画之和（单名+1）
外格 = 总格 - 人格 + 1
总格 = 姓名所有字笔画之和
```

**五行判断**:
- 根据笔画尾数判断五行
- 1/2→木, 3/4→火, 5/6→土, 7/8→金, 9/0→水

## 使用说明

### 改名建议模块
1. 点击导航标签「改名建议」
2. 输入当前姓名、性别、出生日期（可选）
3. 输入想改的名字（每行一个）
4. 点击「分析姓名」按钮
5. 查看详细分析报告（三才五格、五行、音律、寓意、谐音、综合评分）
6. 可导出HTML报告

### 公司取名模块
1. 点击导航标签「公司取名」
2. 选择行业类型、期望字数、偏好风格
3. 可选：输入创始人八字
4. 点击「生成公司名」按钮
5. 查看5个推荐公司名及其分析
6. 点击「收藏」保存喜欢的名称
7. 查看收藏列表、导出或清空

### 手机号增强功能
1. 点击导航标签「手机号」
2. 输入11位手机号，点击「分析手机号」
3. 自动显示「数字风水学分析」（五行分布、调理建议）
4. 输入八字信息，点击「分析匹配度」
5. 查看号码与八字的匹配结果
