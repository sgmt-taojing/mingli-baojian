## 一、四时五行旺衰：旺相休囚死五态（卷一·论五行生克；卷二·论五行旺相休囚死并寄生十二宫）

本库 02 用「得令/失令」二分判旺衰，《三命通会》给出更细的**五态**模型，用于精确衡量某五行在当月的能量层级。

**取法（以月令当令之气为参照）：**

| 状态 | 定义 | 能量层级 |
|---|---|---|
| **旺** | 当令者旺（与月令同气） | 最强，得时秉令 |
| **相** | 我生者相（月令所生，子乘父业） | 次强，得生而进 |
| **休** | 生我者休（生月令者，功成身退） | 平，无事退居 |
| **囚** | 克我者囚（被月令所克之母气受牵制） | 弱，受困不展 |
| **死** | 我克者死（被当令之气所克） | 最弱，受制无气 |

原文以春木为例：「春木旺，旺则生火，火乃木之子，子乘父业，故火相」，水生木而功成则休，金本克木而木旺金反受牵则囚，木所克之土则死（卷二·论五行旺相休囚死并寄生十二宫）。

**四时五行旺衰速查表：**

| 季节（当令） | 旺 | 相 | 休 | 囚 | 死 |
|---|---|---|---|---|---|
| **春**（木令） | 木 | 火 | 水 | 金 | 土 |
| **夏**（火令） | 火 | 土 | 木 | 水 | 金 |
| **秋**（金令） | 金 | 水 | 土 | 火 | 木 |
| **冬**（水令） | 水 | 木 | 金 | 土 | 火 |
| **四季土**（辰戌丑未月、土令） | 土 | 金 | 火 | 木 | 水 |

> 实用要点：
> - 判某神能量层级时，**先定它处五态中哪一档**，比单纯说「旺/弱」更精细。相为进气（有后劲），休为退气（暂安），囚死则需生扶方堪用。
> - 土的旺衰争议大，本表取「四季土寄旺辰戌丑未月」一说；另有「土旺于夏（火土同宫）」与「土寄四立前十八日」诸说，遇分歧以**实际生克对命局有无作用**为准，不纠名目（同 06 神煞分歧处理原则）。
> - 原典提醒：「见旺相不必皆为吉，见休囚死绝不必皆为凶。旺极宜制，死绝不及宜生」（卷二）。**旺衰只是底料，喜忌才定吉凶**，承接 02 用神思路。

## 二、寄生十二宫：体用深化与实用判法（卷二·论五行旺相休囚死并寄生十二宫；论天干阴阳生死）

本库 02 已列长生十二宫的基本对应，本节补《三命通会》的**取象义、阳顺阴逆争议、以及十二宫在断命中的实际用法**。

### 1. 十二宫顺序与取象（卷二原文「长生、沐浴、冠带、临官、帝旺、衰、病、死、墓、绝、胎、养，循环无端，周而复始」）

| 宫 | 取象（人/物之生命周期类比） | 断命常用倾向 |
|---|---|---|
| **长生** | 万物发生向荣，如人始生 | 利创建新事、根基生发、有源 |
| **沐浴** | 形体初成尚嫩，易受损（又名败地、咸池） | 易动摇、感情情色之机、不稳 |
| **冠带** | 渐荣秀、束带成形 | 渐入佳境、自立 |
| **临官** | 物已成材、出仕任事（即禄位） | 兴盛进取、得禄、自立有力 |
| **帝旺** | 物极而盛、壮旺至顶 | 鼎盛、能任事，但盛极转衰之机 |
| **衰** | 物形将衰 | 退气、守成、不宜进取 |
| **病** | 物气衰病 | 滞碍、健康与事多阻 |
| **死** | 物气已尽 | 静止、无力、宜静不宜动 |
| **墓（库）** | 成物收藏入库 | 收敛、贮藏、入墓主收藏或闭塞 |
| **绝** | 前气已绝、后气未续 | 转折点、绝处可逢生、极不稳 |
| **胎** | 后气续、结胎 | 孕育、潜伏、待发 |
| **养** | 胎成将生、养于母腹 | 蓄养待时、承接长生 |

### 2. 阳顺阴逆与「阳生阴死」（卷二·论天干阴阳生死）

原文：「阳生阴死，阳死阴生，循环逆顺，变化见矣。」即**五阳干顺行十二宫、五阴干逆行十二宫**。

- 五阳干长生位（顺行）：甲长生在亥、丙戊长生在寅、庚长生在巳、壬长生在申。
- 五阴干长生位（逆行）：乙长生在午、丁己长生在酉、辛长生在子、癸长生在卯。
- 原典名例：「甲木长生在亥……死于午」「乙木……生于午……死于亥」（卷二·论天干阴阳生死），甲乙同为木而生死宫相反，正是阳顺阴逆之体现。

> **重要存疑与流派分歧（务必标注，不可铁口）**：阴干是否真有独立长生，自古两派。
> - **主张阴干独立长生**（《三命通会》采此，配合纳音、禄马取用）。
> - **主张「阴阳同生同死」**（后世子平多从此，论旺衰只认五行不分阴阳天干，如乙木亦旺于亥子、衰于午）。
>
> 实务建议：**论日主旺衰扶抑，以五行同生同死（看地支藏气与月令）为主流稳妥**（与本库 02 一致）；阴干独立长生宫主要用于查**长生、沐浴（咸池）、墓库、禄位等神煞定位**及部分纳音古法。两套不可混用，用哪套先声明。

### 3. 十二宫在四柱的实用读法

- **看根气**：日主或某神坐长生、临官、帝旺为「得地通根」，坐死绝胎为「无根虚浮」，补充 02 判旺衰的地支维度。
- **看墓库**：财官印食坐墓，主该事象有「收藏、迟发、或闭锁」之象，逢冲开库可能引发（与 06 空亡冲实思路呼应，但墓库未必凶，为用之库逢冲反主发）。
- **看十神配宫**：如食神坐长生主福寿绵长、伤官坐帝旺主才高气盛、官星坐死绝主名位虚浮，结合 04 十神类象与宫位落点交叉印证。

## 三、十神立名本义与细类补充（卷五·论古人立印食官财名义；论正官、论偏官、论印绶、论食神伤官等）

本库 04 已备十神类象表，本节补《三命通会》对十神**命名本义**的解释（有助于理解类象根源），及若干本库未展开的细分。

### 1. 立名本义（卷五·论古人立印食官财名义，原文撮要）

| 十神 | 立名本义（《三命通会》释义要点） | 对断命的启示 |
|---|---|---|
| **印绶** | 生我者如父母有恩荫，「印者荫也，绶者受也」，子孙受其福荫；又喻官有印绶方能理事 | 印主荫庇、凭借、权柄之依托，故为靠山学问 |
| **食神** | 我生者有子孙之义，故立名食神 | 我所生而能受其养，故主福禄、子嗣、享受 |
| **官煞** | 克我者，我受制于人，立名官煞（正者为官，偏者为煞） | 约束我者即官杀，故主管束、名位、压力 |
| **财** | 我克者为我所制用，如人能制用财物 | 我能克而用之，故主妻、财、所掌控之资源 |
| **比劫** | 同我者为兄弟同类 | 同气相扶或相争，故主同辈、竞争、分夺 |

> 启示：十神吉凶不在名目，而在**生克关系（谁生我、我克谁）+ 是否为用**。理解本义可避免望文生义（如「七杀」名凶而制化得宜反主大贵）。

### 2. 正官与七杀的细辨（卷五·论正官、论偏官）

- **正官**：阴阳相配之克（如甲见辛、乙见庚），克而有情，主**循理之贵、名正言顺**。喜财生、印护、身旺任官；忌伤官见官、刑冲破害、官星太重而身弱。
- **七杀（偏官）**：阳见阳、阴见阴之克（如甲见庚、乙见辛），克而无情，主**威权、魄力、亦主祸患**。「有制谓之偏官，无制谓之七杀」，**制化得宜（食神制、印化、刃合）则为权为贵，无制则为攻身之鬼**，倾向伤灾是非。此与 04「七杀为用武贵、为忌伤灾」一致，本节补「制与不制」是关键分水岭。

### 3. 伤官的细类（卷五·论伤官，补 04）

《三命通会》强调伤官需分配置看，不可一概论凶：

- **伤官配印**：伤官旺而有印制，主聪明博学、贵显（上格之一）。
- **伤官生财**：身旺伤官旺而透财，主富、善经营。
- **伤官见官**：多主祸患是非（「伤官见官，为祸百端」），但有特例：**金水伤官喜见官（调候需火）**、**木火伤官亦可适度见官**，需结合 03 调候判断，不可铁口。
- **伤官伤尽**：满局伤官而不见官星，反成清格，主人极聪明、性傲。

> 提示：以上细类与 05 格局成败篇互补，05 讲「成败救应」框架，本节补《三命通会》对单个十神配置吉凶的具体判语来源。

## 四、格局与神煞的集成体系（纲目，非全文）

《三命通会》篇幅极大，其特色在于把**禄命旧法（神煞、纳音、十二宫）与子平新法（十神、格局）熔于一炉**，形成一套可层层查阅的大体系。这里只给**它的组织方式与可回查目录**，具体查法仍以本库 05（格局）、06（神煞）为操作主力。

### 1. 它如何编织体系（理解其结构，便于检索）

1. **以五行生克与十二宫为底层**（卷一至卷二）：先立五行旺衰、纳音、寄生十二宫、合冲刑害的基本盘。

3. **以十神与格局为上层主断**（卷五及以后）：用十神定六亲事象、用格局定贵贱层次，神煞退为佐证。
4. **以六十甲子、诗赋断诀为应用层**（后段诸卷）：把前面理论落到具体日时、具体命例的歌诀断语。

> **使用立场（重要）**：这套体系神煞条目极多，《三命通会》本身也强调主次。本库 06 已明示「先断格局用神，再点神煞」，与此一致。**Agent 推演时仍以扶抑+调候+格局三轴为主**（见 02、03、05），《三命通会》的价值是提供**海量神煞与古法的可查出处**，不宜反客为主逐一套用。

## 五、大运、小运、太岁的取法要点（卷二·论大运、论小运、论太岁、总论岁运）

本库 01-07 偏重原局判读，行运取法少有系统覆盖，本节据《三命通会》补齐。

### 1. 大运（卷二·论大运）

| 要点 | 内容 |
|---|---|
| **来源** | 从月柱起，按出生节气推排干支，「以大运一辰十岁，以三日折除为一年」 |
| **顺逆** | **阳年生男、阴年生女顺行**（以生日后未来节气日时为数，顺数）；**阴年生男、阳年生女逆行**（以生日前过去节气日时为数，逆数） |
| **起运岁数** | 数生日到节气的天数，**三日折一岁、一日折四个月**（一时辰折十日），得起运虚岁 |
| **管辖** | 一运管十年，「大运司十年之休咎」；常以**天干管前五年、地支管后五年**，干支各有侧重（一说干支同参、分轻重） |

> 实用：行大运吉凶看**运干支对原局用神的生扶或克制**，以及与原局的合冲刑会，不单看运支本身吉凶。这是行运判读的核心，承接 02 喜忌体系。

### 2. 小运（卷二·论小运）

- 定义：「大运司十年之休咎，小运掌一岁之灾祥」，小运补大运之不足。
- 起法：「男起丙寅顺行，女起壬申逆行」，自一岁起逐年排一干支。
- 现代用法：**小运在实务中权重远低于大运、流年**，多备而少用，断流年优先取太岁（流年）。本库标注此点，避免 Agent 过度依赖小运。

### 3. 太岁 / 流年（卷二·论太岁、总论岁运）

| 要点 | 内容 |
|---|---|
| **地位** | 「太岁乃一岁之主宰，诸神之领袖」，即当年流年干支，断流年以此为主 |
| **岁运关系** | 「太岁如君，大运如臣」，君臣和悦则其年吉，刑战则其年凶，故**流年与大运需合参**，看二者及与原局的生克合冲 |
| **岁运并临** | 流年干支与大运干支相同（岁运并临），古谓「岁运并临，灾殃立至」，倾向多事之年，**需看其为用为忌再定吉凶**，不可一律断凶 |
| **犯太岁** | 流年支与日支（或年支）相冲、相刑、相同，主**该年多变动、压力、人事波折**，趋势化看待，制化得宜可减 |

### 4. 行运吉凶的十二宫读法（卷二·论大运原文撮要）

原典以十二宫论行运气象，可作流程化判语参考（趋势化）：

- 运到**长生**：倾向有创建作新之事、生机萌发。
- 运到**临官、帝旺**：倾向兴盛进取、利发福进财、添丁之喜。
- 运到**衰、病**：倾向退败、破财、疾病之事，宜守。
- 运到**死、绝**：倾向阻滞、骨肉或自身衰祸、百事蹇塞，宜静守待转。

> 务必叠加喜忌修正：上述只是**十二宫的气象底色**，运之真吉凶仍取决于**运干支对原局用神的作用**（为用则吉、为忌则凶）。十二宫读法是辅，喜忌生克是主，与本篇第一、二节及本库 02 一脉相承。
