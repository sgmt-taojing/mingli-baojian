# 格局：八格取法 · 成败救应 · 特殊格局

> 格局定**贵贱层次与人生格调**，用神定**调节与吉凶**。二者并看：格成而用神有力 → 上命；格破或用神受伤 → 减等。

## 一、八格取法（以月令为纲）

月令（提纲）是格局之源。取格步骤：

1. **月支本气透出天干** → 即以该十神为格。如月支寅，本气甲，甲透天干，日主为庚 → 偏财格。
2. **本气未透，取中气/余气透出者**为格。
3. **藏干皆不透**，则以月支本气十神论格。

八格（以月令所主十神命名）：
- **正官格 · 七杀格**（官杀格）
- **正财格 · 偏财格**（财格）
- **正印格 · 偏印格**（印格）
- **食神格 · 伤官格**（食伤格）

> 月令为比劫（日主临禄、刃）不入八格，另立**建禄格**（月令临官）、**月刃格 / 阳刃格**（月令帝旺），以局中他神取用。

## 二、格局成败（用神顺逆 + 相神护卫）

格局分**顺用**与**逆用**两类，关键看「相神」（护格之神）是否到位：

**顺用之格**（喜生扶流通，财·官·印·食）：
- **正官格**：喜财生官、印护官；忌伤官克官、刑冲。
- **正财格**：喜食伤生财、官星护财；忌比劫夺财、印星泄食。
- **正印格**：喜官杀生印；忌财星坏印（贪财坏印）。
- **食神格**：喜身旺、财星流通（食神生财）；忌枭神夺食。

**逆用之格**（喜制伏，杀·伤·刃·枭）：
- **七杀格**：喜食神制杀、印星化杀、阳刃合杀；忌财党杀攻身、杀无制。
- **伤官格**：喜佩印（伤官配印）、生财（伤官生财）；忌见官（伤官见官，为祸百端，除金水/木火特例）。
- **阳刃格**：喜官杀制刃；忌刃无制而冲提。
- **偏印格**：喜财制枭、官杀生身；忌枭神夺食、印重身滞。

**成格**：用神得相神护卫、配合得宜，全局流通。
**破格**：忌神坏用（如官逢伤、印逢财、食逢枭、杀无制）。
**救应（复成）**：破格之后另有一神制去忌神或合化解之，则败而复成，反主历险成事。

口诀：**成中有败，必是带忌；败中有成，全凭救应。**

## 三、特殊格局（不入常格，顺其旺势）

当日主或某神**极旺成势、无法逆制**时，弃命相从或从旺：

### 从格（日主极弱无根无生，舍己从人）
- **从财格**：满盘财星，日主无根无印比 → 从财，喜财食、忌印比。
- **从杀（官）格**：满盘官杀克身无制无印 → 从杀，喜财官、忌食伤印比。
- **从儿（食伤）格**：满盘食伤泄秀，日主虚弱 → 从儿，喜食伤财、忌印。
- **从势格**：财官食伤并旺、无一可舍 → 从其最旺之势。
- 成从条件极严：原局有一点通根或得力印比生扶，则**不成从，按身弱扶抑论**。误判从格是大错。

### 专旺格（一行独旺，顺其气势）
- **曲直格**（木）、**炎上格**（火）、**稼穑格**（土）、**从革格**（金）、**润下格**（水）。
- 全局同一五行成方成局、无克破，日主即旺神 → 顺用：喜比劫、印、食伤泄秀，忌官杀克、财耗。

### 化气格（日干合化）
- 日干与邻干五合（甲己·乙庚·丙辛·丁壬·戊癸），化神得月令、四柱拱扶，真化 → 论化神旺衰取用。
- 化神不真（化神失令、有争合妒合）则按未化论，常为忌。

### 其他常见特殊格
- **魁罡格**（庚辰·庚戌·壬辰·戊戌日）：聪明刚毅、掌权，喜身旺无冲刑，忌财官填实、刑冲。
- **金神格 · 井栏叉 · 六阴朝阳**等古格，需严格条件，从略，现代实战以**扶抑 + 调候 + 格局成败**三轴为主即可定命。

## 四、定层次（高低）三问

1. **格是否成？** 相神是否到位、有无破败、破后有无救应。
2. **用神是否有力？** 透干通根、得令得地，还是虚浮受制。
3. **全局是否流通中和？** 五行不缺不偏、生克有情、寒暖得宜 → 上格；偏枯、战克、调候不及 → 减等。

格高用旺 → 富贵显达；格成用弱或格破有救 → 中等、起伏后成；格破无救、用神受伤 → 平常或坎坷。
