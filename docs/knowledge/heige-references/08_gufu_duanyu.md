# 古籍赋文断语精选：经验规则 · 短引 · 白话 · 调用提示

> **这是经验断语规则库，供"一象多看"取佐证用，孤证不立。** 历代赋文多为前人验命口诀，言简意赅但常以极端笔法立言（"定""必""无疑"之类），便于记诵，不等于铁口。每条断语都须回到本命的**旺衰、用神、格局成败**去核（参 02 旺衰五看 / 03 穷通调候 / 05 八格成败）。同一句赋文，换个旺衰背景结论可反。本文只录 01-07 未覆盖的"经验断语"，按人生维度归类，每条给原文短引、白话解、现代调用提示与出处。出处以《渊海子平》《三命通会》为主，均属公版，原句经 ctext.org、古诗文网、维基文库等核对；个别赋名或归属有争议者标"存疑"。

## 凡例

- **出处简记**：`渊·喜忌篇` = 《渊海子平》喜忌篇；`通·继善篇` = 《三命通会》卷十一继善篇；`通·络绎赋` = 卷十二络绎赋；余类推。同一名句两书互见者并列。
- **断语口径**：全部转为趋势语（易 / 倾向 / 利于 / 需注意），原文的绝对句式仅作短引保留，不作承诺。
- **调用提示**：写给 AI Agent 看，指明这条规则在推演时"先验什么、再下什么"，避免单句直断。

## 一、总论纲领

定盘先后、取象主次的总则。赋文开宗多讲"先看什么"，是排盘后第一步的次序锚点。

| 原文要点（短引） | 白话解 | 现代调用提示 | 出处 |
|---|---|---|---|
| 欲知贵贱，先观月令乃提纲；次断吉凶，专用日干为主本 | 论格局高低先看月令（提纲），断吉凶以日主为核心 | 推演次序：月令定格→日主定强弱→再看十神配置。勿一上来就抓神煞或单字 | 渊·继善篇 / 通·继善篇 |
| 三元要成格局，四柱喜见财官 | 天地人三元配合宜成格，财官为常人所喜的用神 | 普通命造以财官为荣身之物；但身弱不胜财官时反为忌，须先验旺衰 | 渊·继善篇 |
| 有杀只论杀，无杀方论用 | 命带七杀先处理杀（制或化），无杀再论其他取用 | 见七杀透出且成势，优先看有无食制、印化、刃合；七杀未安顿，别急着论财论印 | 神峰·五言独步 |
| 有病方为贵，无伤不是奇；格中如去病，财禄两相随 | 命局有"病"（偏枯之处）而岁运能去病者，反主大用 | 不必见病就判凶。先找病（偏枯 / 忌神当令），再看原局或大运有无"药"；药到则吉应。详见 09 病药说 | 神峰·病药说 |
| 衰运发而旺运止，旺运发而衰运终 | 用神所喜之运到则发，所忌之运到则止 | 应期判断的总纲：发与止系于运的喜忌方向，不系于运本身"旺不旺" | 通·络绎赋 |
| 富莫富于纯粹，贫莫贫于战争；贵莫贵于秀实，贱莫贱于反伤 | 五行清纯不杂主富，干支交战刑冲主贫；秀气流通主贵，用神受伤主贱 | "纯粹 / 秀实"对应五行流通无阻；"战争 / 反伤"对应刑冲克破。先看全局清浊再定层次高低 | 通·爱憎赋 |

**纲领提醒**：赋文里的"贵 / 贱 / 富 / 贫"是古代社会语境下的极化表述，现代调用一律转成"层次高低、顺逆倾向、资源厚薄"，不照搬阶级判词。

## 二、性格心性

十干配合与五行交战常被用来取性情。这一维度赋文给的是**性格倾向的取象线索**，宜与十神性情（参 04 十神六亲）叠看，不单据一句定性。

| 原文要点（短引） | 白话解 | 现代调用提示 | 出处 |
|---|---|---|---|
| 甲逢己而生旺，定怀中正之心 | 甲木见己土（甲己合）且生旺，性情多端正中和 | 取"合而得用"为性格安定之象；若合而被冲破，则中正之性打折 | 通·继善篇 |
| 乙遇庚而旺，常存仁义之心 | 乙木得庚金（乙庚合）成器而旺，多有仁义担当 | 合化成金为用时取此象；合化不成则仅作牵绊看 | 通·继善篇 |
| 金木交争刑战，仁义俱无 | 金木相战刑冲不解，仁（木）义（金）两失，性多偏激 | 见金木对冲无通关（水）者，提示情绪冲突、易走极端；有水通关则解 | 通·继善篇 |
| 丁遇壬而太过，必犯淫讹之乱 | 丁壬合木太过（合多 / 化神泛滥），易感情用事、是非牵缠 | "丁壬淫匿之合"的取象：合多者重情、易陷感情纠葛。仅作倾向，不作品行定论 | 通·继善篇 |
| 火木通明，见土则贵；金水相涵，笔下千篇锦绣 | 木火相生而通明者聪秀，金水相涵者文思敏捷 | 食伤生发、五行流通为"聪明秀气"之象，可佐才华、表达、学术倾向 | 通·金声玉振赋 |
| 曲直兼资乎印绶，仁声播九有以无穷 | 木命成曲直格又得印绶相生，仁德声名远播 | 专旺格（曲直 / 炎上等）配印生扶时，性格取"格局之性"为主（参 05 专旺格） | 通·金声玉振赋 |

**心性提醒**：性格断语极易被滥用为"贴标签"。调用时只输出"倾向于……""遇某运易……"，并提示这是多因素之一，须结合十神格局综合。

## 三、事业 · 官杀印

官杀主事业地位与压力承受，印主庇护、文凭、贵气。赋文在此最密，核心是**官杀的制化与身杀平衡**。

| 原文要点（短引） | 白话解 | 现代调用提示 | 出处 |
|---|---|---|---|
| 时逢七杀，见之未必为凶；月制乾强，其杀反为权印 | 七杀本凶，若月令有制、日主强旺，则化杀为权 | 见七杀勿径判凶。先验：日主是否够强、有无食制 / 印化 / 刃合。制化得宜则主权柄、魄力、担当 | 渊·喜忌篇 |
| 四柱纯杂有制，定居一品之尊；略有一位正官，官杀混杂反贱 | 七杀有制可贵，但官杀并见而混杂无理则降格 | 先分清官杀是否混杂；混杂须有"去官留杀"或"去杀留官"之机方清。混而不清提示事业反复、定位不明 | 渊·喜忌篇 |
| 官星得令，制伏诸凶；贵人扶身，解脱百厄 | 正官当令得力，可压制群凶；逢贵人则逢凶化吉 | 官星为用且得令时，事业根基稳；再叠天乙等贵人（参 06 神煞）则助力明显 | 通·金声玉振赋 |
| 印旺官生，必掌钧衡之任 | 印星旺而有官来生印，多任重要、掌实权 | "官印相生"为贵格之一。验官能生印、印能护身且为用，提示管理、权责、体制内发展之利 | 通·明通赋 |
| 官轻不若杀轻，所以喜财旺之地 | 官星偏轻时，行财运生官，事业方起 | 官弱用财生官为常法；但身弱者财旺反助官克身，须先过旺衰这关 | 通·明通赋 |
| 杀刃两现干头，定受言责之寄 | 七杀与羊刃并透天干（杀刃格），多任威权、纪律、生杀之职 | 杀刃格取"以刃当杀、以杀制刃"之平衡，利武职、执法、竞争性强的领域；忌刃逢冲（参 06 羊刃） | 通·金声玉振赋 |
| 日禄居时，没官星，号"青云得路" | 日主之禄落时支而四柱不见官星（归禄格），主自立显达 | 归禄格忌官星填实破格。见此结构提示靠自身能力发达，宜核有无官来破局 | 渊·喜忌篇 |

**事业提醒**：官杀类断语的开关全在"身能否任"与"杀有无制"。AI 推演时务必先输出旺衰结论，再套用本节规则，否则身弱见杀误判为权、身强无制误判为凶，皆属颠倒。

## 四、财运 · 食伤财

财为养命之源，食伤为生财之机，亦为泄秀、伤官之口。此维度核心是**财印相战、食伤生财、伤官见官**三组关系。

| 原文要点（短引） | 白话解 | 现代调用提示 | 出处 |
|---|---|---|---|
| 财生官，官生印，印生身，富贵双全 | 财→官→印→身一路相生，财官印三者循环有情，主富贵 | 见此连环相生且不断链、不被冲破者，提示财业与地位兼得。先验链条有无断点（如印被财坏） | 通·络绎赋 |
| 伤党财，财党杀，杀克身，凶穷两逼 | 伤官助财、财助杀、杀来克身，一路克身无制，主困顿 | 与上条相反的连环克身结构。见食伤生财、财又生杀而身弱无救者，提示破财兼伤身，需印比解救 | 通·络绎赋 |
| 财坏印者，贪财坏（神职） | 财星破印（财印交战，财胜），多因贪财而损名位、误正事 | 印为用而财来坏印时，行财运反主是非、失位。提示"重利轻名"之险，宜比劫制财或官星通关 | 通·络绎赋 |
| 财官印绶全备，藏蓄于（辰戌丑未）四季之中 | 财官印同藏于四墓库（辰戌丑未），是力量库藏 | 见四库藏财官印者，须看库能否被冲开 / 透干引出；藏而不发则富贵在内难显（参 01 四土库、藏干） | 渊·喜忌篇 |
| 印绶生月，岁时忌见财星 | 月令印绶为用者，岁运与时柱忌见财来坏印 | 印格用印时，财为忌神。逢财年财运提示破耗、长辈或文书之扰。先确认印确为用神再断 | 渊·喜忌篇 |
| 时上偏财，怕逢兄弟 | 时柱偏财为用，最怕比劫（兄弟）来分夺 | 时上偏财格逢比劫岁运，提示破财、合伙生隙、与同辈争利。身弱另当别论 | 通·继善篇 |
| 伤官见官，为祸百端 | 伤官格而原局或岁运见正官，伤官克官，多生祸端 | 经典烈语，须分流派：**伤官伤尽 / 火土伤官**忌见官；**金水伤官、伤官佩印 / 伤官生财**则见官未必为祸。调用时先判伤官属哪一路，不可一律判凶 | 渊·喜忌篇 / 通·论伤官（经云） |
| 用财者制人，用官杀者制于人 | 以财为用者主动掌控（制人），以官杀为用者受制于位（制于人） | 取象差异：财格之人多自主、经营性强；官杀格之人多在体制 / 受约束的结构中显达。供事业财运取向的参考 | 通·金声玉振赋 |

**财运提醒**：财能生官亦能坏印，食伤能生财亦能伤官，方向全看用神。"伤官见官为祸百端"是最易被误用的一句，金水伤官、伤官佩印的命见官反喜，务必先定伤官类型再下语。

## 五、婚姻 · 六亲

赋文论六亲多借十神宫位（财为妻、官杀为夫、印为母、比劫为兄弟）与孤寡神煞取象。此处只补"赋文经验断语"，宫位与十神对应详见 04 十神六亲。

| 原文要点（短引） | 白话解 | 现代调用提示 | 出处 |
|---|---|---|---|
| 孤寡者，只为财神被劫 | 命见孤辰寡宿之孤，多因财星（男以财为妻）被比劫劫夺 | 男命财为妻星，财被劫近重者，提示婚姻聚少离多、配偶缘较薄。须叠神煞与财星状态同看（参 06 孤辰寡宿） | 通·元理赋 |
| 凡人命犯孤寡，主形孤骨露，不利六亲；生旺稍可，死绝尤甚 | 带孤辰寡宿者六亲缘偏淡，逢生旺减轻，逢死绝加重 | 孤寡为辅断之煞，不主断。仅在六亲星本就受损时作加重 / 减轻的修饰，单见不取 | 通·论孤辰寡宿 |
| 时上偏财，怕逢兄弟 | （同财运章）时上财为用逢比劫，亦应于六亲：与兄弟、合伙人易争利 | 既看破财，也看人际：提示与同辈、伙伴在利益上易生嫌隙 | 通·继善篇 |
| 财官双美，透露极荣 | 财与官皆清而透干得用，主婚姻事业两旺（男命财官并美） | 男命财官两清且为用，提示得力配偶兼事业上行；须先排除财官混杂、争合 | 通·金声玉振赋 |
| 女人无杀带二德，作两国之封 | 女命不见七杀混杂、又带天德月德者，多贤淑得贵 | 古代女命以"夫子"为重的语境产物。现代只取"官星清、德煞助"为婚姻家庭和顺之倾向，不沿用从夫贵的判词 | 通·继善篇 |

**六亲提醒**：女命赋文带浓厚旧时代色彩（重夫从夫、以子为贵），调用时一律剥离价值判断，只保留"某星清浊→某段关系顺逆"的结构信号。婚姻吉凶尤须忌单凭一句神煞或一颗孤星直断。

## 六、健康

赋文论疾病以五行受克、太过不及定病位，是少见的"病象"专门口诀，可作健康倾向的辅助提示。**仅供养生与体质倾向参考，不作医疗诊断**。

| 原文要点（短引） | 白话解 | 现代调用提示 | 五行 / 病位 | 出处 |
|---|---|---|---|---|
| 生平少病，日主高强 | 日主旺而有根，体质底子多较强健 | 日主得令通根者，整体抗病力佳；身弱无根者反之 | 体质总论 | 通·继善篇 |
| 金弱遇火炎之地，血疾无疑 | 金（肺、大肠）弱而火旺克金，易见呼吸、血分之疾 | 金为忌受火克且无土通关时，提示肺、气管、血气方面需留意 | 金：肺 / 血 | 通·继善篇 |
| 土虚逢木旺之乡，脾伤定论 | 土（脾胃）虚而木旺克土，脾胃易弱 | 土弱被木克重者，提示消化、脾胃系统较敏感，宜饮食调护 | 土：脾胃 | 通·继善篇 |
| 筋疼骨痛，皆因木被金伤 | 木（肝胆、筋）被金克过重，易见筋骨、肝胆不适 | 木为忌受金克且无水化时，提示筋骨、肝胆、神经方面需注意 | 木：肝 / 筋 | 通·继善篇 |
| 眼暗目昏，必是火遭水克 | 火（心、眼目）被水克过重，易见心血、眼目之患 | 火弱被水克重者，提示心血管、视力方面留意；调候喜火者尤甚（参 03 调候） | 火：心 / 目 | 通·继善篇 |
| 丙临申位逢阳水，难获延年 | 丙火坐申又逢壬水重克（火极弱），主健康根基较弱 | 此类专指特定干支组合的极端象，仅在火确为命脉用神且严重受克时作提示，不可见组合即断寿 | 火极受克 | 通·继善篇 |

**健康提醒**：五行配脏腑是中医取象的延伸，存在体质倾向与岁运诱发的统计经验，但赋文用"无疑""定论""难获延年"等绝对语属记诵笔法。AI 调用必须降级为"需留意 / 宜调护"，明确这不是诊断，并建议就医以专业为准。

## 七、吉凶与应期

应期是"何时应"的判断，赋文核心是**岁运与命局的生克冲合、用神到位与否**。配合 05 格局成败、06 神煞冲合一起用。

| 原文要点（短引） | 白话解 | 现代调用提示 | 出处 |
|---|---|---|---|
| 日犯岁君，灾殃必重；岁伤日干，有祸必轻 | 日干去克太岁（流年干）为"犯"，祸较重；太岁来克日干，祸较轻 | 流年与日主的克向不同，应力轻重有别。下克上（日犯岁）应力重于上克下，作流年吉凶的方向参考 | 通·继善篇 |
| 五行有救，其年反必为财 | 凶年若命局有救应（通关、制化之神），反可转祸为利 | 判流年凶象前，先找原局与大运有无"救神"。有救则凶象减、甚至转机；无救方论实凶 | 通·继善篇 |
| 月生日干，运行不喜财乡 | 印绶生身为用者（印格），行运不喜走财地（财坏印） | 印为用之命，财运为忌，提示该阶段易有破耗、文书长辈之扰。与"印绶生月忌见财"互证 | 通·继善篇 |
| 日主无依，却喜运行财（官）地 | 日主有依无依决定喜忌方向相反（此为身旺无依、喜泄喜克之意） | 强调"同一运的吉凶随旺衰反转"。身旺无依者喜财官泄克，身弱无依者喜印比生扶，方向须先定 | 通·继善篇 |
| 衰运发而旺运止，旺运发而衰运终 | （同总论）发与止取决于运的喜忌，不取决于运强弱 | 应期总开关：先定用神，再看大运流年是助用还是助忌，助用则发、助忌则止 | 通·络绎赋 |
| 空亡 / 冲实见应 | 落空之神逢冲或填实之年方应（参 06 空亡） | 原局逢空的吉凶之神，多在岁运冲空、填实时应事。空亡之应期专靠这把钥匙 | 通·论空亡（辅证） |

**应期提醒**：应期最忌单看流年一字。标准流程：① 先定原局用神喜忌 → ② 看大运是顺是逆 → ③ 流年与大运、命局的生克冲合在哪一柱触发 → ④ 神煞、空亡作落点修饰。四步齐了再给"某年某事易……"的趋势语。

## 八、使用原则（务必遵守）

1. **孤证不立，一象多看**。任何一条赋文断语都只是一条线索，须再找两到三个同向信号（旺衰、用神、格局、十神、宫位、神煞）相互印证，方可输出倾向；单句直断一律不取。
2. **先定旺衰用神，再套赋文**。本文绝大多数断语的吉凶方向都随日主强弱反转。推演时务必先有旺衰结论（参 02），再调用本节规则，否则极易把"身弱见杀"误判为权、把"金水伤官见官"误判为祸。
3. **极端语降级为趋势语**。赋文"定""必""无疑""为祸百端""难获延年"皆属记诵笔法。对外输出统一改为"易 / 倾向 / 利于 / 需注意 / 宜调护"，不承诺祸福、不下死语。
4. **分清流派与适用前提**。如"伤官见官为祸百端"须先判伤官类型（火土忌见官、金水喜见官、佩印生财各异）；"有杀只论杀"须先确认杀已成势。脱离前提套用即误。
5. **健康、婚姻、寿元类最审慎**。健康断语只作体质倾向与养生提示，并申明非医疗诊断；婚姻六亲断语剥离旧时代价值判断，只取结构信号；涉寿元的极端组合一律不单独成断。
6. **古今语境转换**。赋文中的"贵贱、富贫、封诰、僧道"是古代社会语境，调用时转为"层次高低、资源厚薄、发展取向、宜独立 / 宜专研"等中性表述。
7. **存疑标注，不杜撰**。本文短引均经公版库核对；个别赋名归属（如《万金赋》的可靠公版底本未及确认）暂未收录或标"存疑"，宁缺勿造。遇未核实的"古文原句"不得编造引用。
