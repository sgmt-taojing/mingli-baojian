# 可视化命书 · 一页 HTML 报告产出规范

> 文字推演（SKILL 第 0–10 步）是根，可视化只是把结论**呈现**得更易读、易藏、易分享。先有完整推演，再谈可视化；HTML 内容与文字版逐字一致，绝不为排版另造结论。

## 一、何时做 · 先征询

- 推演全部给完后，主动问用户一句：要不要把这份命书做成一页可视化 HTML（便于保存、回看、分享）。用户明确要才做。
- 动手前确认两件事：① 报告主题（默认「我的一生」）；② 信息口径，分享场景是否隐去真实姓名与生辰（脱敏）。
- 没有完整的第 0–10 步，不直接出 HTML。可视化是锦上添花，不替代推演的严谨。

## 二、设计定位

- 一页到底的「命书」长卷：东方雅致、有命理质感，像一本可收藏的个人之书。
- 一屏一主角：每屏只讲一件事（一个命盘、一组五行、一条主轴结论），不堆砌。
- 签名时刻：全局设计一个最大记忆点，把命局命门（最关键的用神、缺神或格局字）单字放大成全屏主视觉。

## 三、字体可靠性铁律（最重要，踩过坑）

中文命书最容易翻车在字体，规则按优先级：

1. **思源宋体为骨架，系统宋体兜底**：正文与标题统一字体栈 `'Noto Serif SC','Songti SC','STSong','Songti TC',serif`。webfont（Noto Serif SC）联网时加载，断网或加载失败时无缝退到系统宋体，断网也不塌字、绝不掉成 Times。
2. **层次靠字重，不靠书法字体**：大标题与签名字 `font-weight:900`、命盘单字 700、雅致小标 400–600、正文 300。Google Fonts 的 `<link>` 记得把 `900` 加进 `wght@…;900`。
3. **不赌本地书法体**：魏碑、行楷、楷体（Weibei SC / Xingkai SC / Kaiti SC）在 macOS 多是「按需下载」字体，未必预装，写进字体栈会塌成默认衬线。想要书法效果只能内嵌子集化字体文件，否则一律用宋体加字重方案。
4. **拉丁数字与年份**：用一款衬线（如 Cormorant Garamond），兜底 Georgia，与中文宋体协调。
5. **大标题防孤字**：`white-space:nowrap` + `clamp()` 控字号 + `text-wrap:balance`；中文宽度不要用 `ch` 单位。

> 验收：断网打开页面，中文应仍是工整宋体而非 Times 衬线；标题字重层次分明。

## 四、性能与反 AI 体检（出货铁律）

- 不动画 `text-shadow` 与 `box-shadow`；不用 `backdrop-filter`；不给带模糊的元素做动画。
- 无限动画只动 `transform` 和 `opacity`；不留常驻 `requestAnimationFrame`。
- 所有动效能被 `@media (prefers-reduced-motion:reduce)` 一键关停。
- reveal 揭示用 IntersectionObserver；无 JS 时内容直显（`.reveal` 默认可见，挂上 `.js` 类才隐藏起始态）。
- 反 AI 套路：拒绝紫色渐变白底、Inter 字体、居中堆叠的模板感。用纸墨配色加一个朱砂、一个金，疏密与留白要有节奏。

## 五、结构模板（按命局裁剪，非全部必有）

| 区块 | 内容 | 取自推演 |
|---|---|---|
| 卷首 Hero | 主题标题、命主、生辰、一首命格诗（4 句点出命局核心） | 总评定性 |
| 一图读懂 | 3–5 张关键结论卡片 | 各维度核心结论 |
| 命盘全图 | 四柱表：干支 / 十神 / 藏干 / 星运 / 纳音 / 旬空 | 第 1 步排盘 |
| 五行能量 | 五行个数能量条，缺失五行高亮 | 脚本五行分值 |
| 旺衰用神 | 身强弱一句定档 + 用 · 喜 · 忌 | 第 3–4 步 |
| 签名时刻 | 命门一字全屏放大（用神或缺神），辅以命局意象 | 用神 / 调候 |
| 格局 | 取格 + 成败救应 + 层次 | 第 5 步 |
| 大运流年 | 时间轴：各步大运干支、十神、吉凶；近十年流年 | 第 6 步 |
| 分维度详断 | 性格 / 事业 / 财运 / 婚姻 / 健康 / 学业 / 六亲卡片 | 第 8 步 |
| 趋避 | 方位 / 颜色 / 行业 / 注意事项 | 第 9 步 |
| 色彩服饰 | 衣着 / 首饰 / 配饰 / 随身 / 环境的色彩材质，认用神色、宝石按色参考 | 第 9 步 · `references/16` |
| 养生调养 | 作息 / 饮食 / 情志 / 运动，按用神喜忌与寒暖燥湿定制 | 第 9 步 · `references/15` |
| 总评 | 一生节奏 + 趋吉关键词 | 第 10 步 |
| 页脚 | 「仅供研究参考」声明 | 合规 |

每个断语块保留一枚「依据标签」（如 `伤官透月干为用`），与文字版推理链一致，做到可视化也带依据。

## 六、配色取命局

- **主色调取用神五行**：木青绿 · 火朱红 · 土土黄 · 金月白 · 水墨蓝。用神色做全局基调。
- **底色**：墨黑或宣纸暖白二选一，深色更显命理质感。
- **签名色**：一个朱砂（印章、书脊、强调）加一个金（标题、分隔线、边框）。
- **忌神色仅作点缀**：例如火为忌，火红只用在「软肋 / 警示」处，不铺满。
- 一份报告的色彩控制在「底色 + 用神主色 + 朱砂 + 金 + 一个辅助色」之内，不堆色。

## 七、脱敏与合规

- 对外分享或入库（如做成 skill 示例）：八字加生辰本身就是强身份标识，**只改姓名远远不够**。要脱敏就换一个虚构生辰、重新排盘，断语写通用教学向，不写可对号入座的具体生活事件。
- 页脚固定声明：本报告是传统文化研究与自我认知参考，非预言或保证，不承诺改运消灾。
- 全篇措辞趋势化（易 / 倾向 / 利于 / 需注意），不恐吓、不承诺祸福，与文字版同一口径。
