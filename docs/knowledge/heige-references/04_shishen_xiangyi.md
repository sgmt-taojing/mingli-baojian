# 十神类象 · 宫位 · 六亲 · 分维度断法

> 用神定吉凶**层次**，十神定具体**事象**。断性格、事业、财婚、健康，皆从十神类象 + 宫位落点展开。

## 一、十神类象（吉凶随其为用神/忌神而转）

| 十神 | 核心象义 | 为用（正面） | 为忌（负面） | 主六亲 |
|---|---|---|---|---|
| **正官** | 约束·名位·责任 | 自律守法、名誉地位、管理力 | 拘谨刻板、压力、官非 | 女命夫；男命女儿 |
| **七杀** | 权威·压力·魄力 | 杀伐决断、武贵、开创、抗压 | 暴躁、小人、伤灾、偏激 | 女命夫/情人；男命儿子 |
| **正印** | 学问·庇护·慈爱 | 学历、贵人、靠山、宗教向善 | 依赖、惰性、不务实、恋母 | 母亲；学业靠山 |
| **偏印（枭）** | 偏学·孤思·机敏 | 玄学技艺、独特智慧、直觉 | 孤僻、多疑、夺食伤福、虎头蛇尾 | 继母、祖辈 |
| **正财** | 正资·务实·节俭 | 稳定财源、勤恳、责任心 | 吝啬、为财所累、平庸 | 男命妻；父亲 |
| **偏财** | 横财·商机·交际 | 经商之才、慷慨、机遇、人脉 | 风流、破耗、投机、不专 | 男命父/情人 |
| **食神** | 才华·福禄·涵养 | 艺术口才、福气、温和、长寿 | 慵懒、好逸、空想 | 女命子女；男命下属 |
| **伤官** | 秀气·叛逆·表现 | 聪明才艺、创新、表达力强 | 恃才傲物、克官非、冲动招怨 | 女命儿子；晚辈 |
| **比肩** | 自我·独立·同侪 | 主见、毅力、合伙、自尊 | 固执、争财、孤行 | 兄弟、同辈、朋友 |
| **劫财** | 竞争·冒险·夺取 | 行动力、社交、敢拼 | 破财、争夺、刚愎、口舌 | 兄弟姐妹、对手 |

口诀：官杀看责任与压力，印星看学识与靠山，财星看务实与机遇，食伤看才华与表达，比劫看自我与人际。

## 二、四柱宫位（落点定应期与领域）

| 柱 | 代表人物 | 人生阶段 | 领域 |
|---|---|---|---|
| **年柱** | 祖辈、父母、长辈 | 1–16 岁、童年 | 根基、出身、家世 |
| **月柱（提纲）** | 父母、兄弟 | 17–32 岁、青年 | 事业根本、社会环境、心性，**最重要** |
| **日柱** | 自身（日干）、配偶（日支） | 33–48 岁、中年 | 自我、婚姻、内在 |
| **时柱** | 子女、下属 | 49 岁后、晚年 | 成就归宿、子嗣、晚景 |

**日支为「夫妻宫」**，看配偶与婚姻；时柱看子女缘与晚运。某十神坐某宫，即该领域显该十神之象（如财坐日支，妻贤助财；伤官在时，子女个性强或晚年多变）。

## 三、六亲取用星

| 关系 | 男命 | 女命 |
|---|---|---|
| 父亲 | 偏财 | 偏财 |
| 母亲 | 正印 | 正印 |
| 配偶 | 正财（妻） | 正官（夫，七杀为情/再婚） |
| 子女 | 官杀 | 食伤 |
| 兄弟姐妹 | 比劫 | 比劫 |

看某六亲：先看其星的旺衰喜忌，再看其所居宫位有无刑冲合害。星旺逢生且为喜用 → 该亲缘厚助力大；星弱受冲克或为忌 → 缘薄或有刑伤。

## 四、分维度断法

### 性格
日主五行阴阳定底色（木仁·火礼·土信·金义·水智；阳刚阴柔），叠加**格局十神 + 月令 + 最旺之神**修正。
- 官杀旺：自律、守规、重责任（过则压抑）。食伤旺：聪慧、外向、爱表达（过则任性）。
- 印旺：内敛、好学、念旧（过则依赖）。财旺：务实、重利、勤快（过则功利）。
- 比劫旺：独立、好强、讲义气（过则固执争胜）。
- 身强者主观能动强、身弱者随和被动；用神得力者格局正、忌神当道者偏激。

### 事业 · 财运
- **事业看官杀与食伤**：官杀为用宜从政、管理、公职、靠组织；食伤为用宜技艺、创作、自由职业、表达型行业；印为用宜文教、研究、医卜。
- **财运看财星 + 食伤生财 + 身财两停**：财为用且身能任财（身强财旺）→ 富；身弱财旺（财多身弱）→ 富屋穷人，逢比劫帮身之运方得财。食伤生财者善经营。财星逢比劫冲夺、入墓受刑者，财来财去。
- **行业方位取用神五行**：用神属木→文教/木材/植物/东方；火→能源/电子/餐饮/南方；土→地产/建筑/农牧/中部；金→金融/机械/五金/西方；水→贸易/物流/水产/北方。颜色、数字同取用神五行。

### 婚姻 · 感情
四看：① 夫妻星（男财女官）旺衰喜忌；② 夫妻宫（日支）受生扶还是刑冲；③ 配偶星与日主合还是冲；④ 桃花、红艳等情缘星。
- 夫妻星为喜用且不受伤、日支为喜用神 → 婚姻美满、配偶得力。
- 夫妻星为忌、或日支逢冲（如子午冲日支）→ 婚姻波动、聚少离多。
- 男命财多或财弱比劫旺、女命官杀混杂或伤官旺克官 → 感情易生波折，需具体看制化。

### 健康
五行配脏腑：**木主肝胆/筋/眼、火主心/小肠/血/神志、土主脾胃/肌肉、金主肺/大肠/呼吸/皮肤、水主肾/膀胱/泌尿/骨**。
- 某五行**过旺**（自身亢）或**过弱被克绝**（受制太过），对应脏腑易出问题。
- 如水弱火炎 → 心肾不交、泌尿；木旺克土 → 脾胃；金弱火旺 → 呼吸皮肤。
- 用神受伤之运为疾厄高发期。

### 学业 · 才华
- 印星（尤正印）为学历、记忆、根基；食伤为才艺、悟性、表达；官印相生利仕途功名；伤官配印为聪明博学之上格。
- 文昌、华盖、学堂助文采；印星为用逢生扶之运利考学。

## 五、断语原则

1. **一象多看，交叉印证**：一个结论至少两处佐证（星 + 宫、或 原局 + 大运），孤证不立。
2. **吉凶看喜忌、轻重看力量、应期看岁运**：是何事看十神，吉或凶看其为用为忌，何时应看大运流年引动。
3. **生克制化是动态的**：忌神被合被制则减凶，用神被冲被夺则减吉。
