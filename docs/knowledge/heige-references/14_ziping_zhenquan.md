# 子平真诠精要：月令取格 · 顺用逆用 · 相神救应 · 成败高低 · 行运同看

> **出处**：《子平真诠》，清·沈孝瞻著（乾隆进士），民国·徐乐吾作评注（通行本即《子平真诠评注》）。本书把子平格局论讲到系统化，自月令立格起，经顺逆取用、相神护格、成败救应，到定层次高低、配气候、论行运，环环相扣，是格局派最完整的理论骨架。
>
> **与本库关系**：本库 05 的整套方法论（月令取格、顺用逆用、相神、成败救应）正是出自《子平真诠》，但 05 偏速查口诀、未锚定原文。本篇补 05 所缺的精密逻辑与原文锚点，尤其是子平真诠独有的「相神」概念，以及成败之中再分「因成得败、因败得成」的辩证层。两篇配套用：判格局先查 05 八格速查，断成败高低回本篇核原文之理。用神扶抑基础在 02，调候在 03，十神类象在 04，神煞在 06。
>
> **断语口径**：以下皆趋势化参考（易 / 倾向 / 利于 / 需注意），非铁口祸福。短引名句加引号照录以便检索，长篇古文不照抄。凡原文逐字者标「原文」，凡撮意转述者标「（大意）」。

## 一、用神总纲：八字用神，专求月令（论用神）

子平真诠立论第一句，原文：**「八字用神，专求月令。以日干配月令地支，而生克不同，格局分焉。」**

理解要点：此处「用神」指**月令所主的那个十神（即格局之神）**，与本库 02 讲的「扶抑用神（调节命局平衡的喜用之神）」是两个不同概念，务必分清。沈孝瞻的「用神」是取格之神，问的是「这命是什么格」；02 的用神是治病之神，问的是「这命喜什么字」。两者在子平真诠体系里通过顺逆与相神接起来。

取格落点（参 05 第一节，本篇不重复列举步骤）：以日干去配月令地支的藏干十神，本气透干即取本气为格，本气不透则取透出的中气、余气，皆不透则仍以月支本气论格。

## 二、顺用与逆用：用神之善与不善（论用神，核心）

这是子平真诠取用的总开关。原文（经 ctext《子平真詮評註》底本核实）：**「财官印食，此用神之善而顺用之者也；煞伤劫刃，用神之不善而逆用之者也。」**配套句：**「当顺而顺，当逆而逆，配合得宜，皆为贵格。」**（大意，通行本作此）

把握两点：一是善神顺用、恶神逆用；二是顺逆用对了、配合得当，任何格都能成贵格，格无高下之分，分在配合。

| 类别 | 用神 | 原文定性 | 取用大方向 |
|---|---|---|---|
| **善而顺用** | 财、官、印、食 | 「用神之善而顺用之者也」 | 喜生扶护卫，使其旺而流通 |
| **不善而逆用** | 煞、伤、劫（阳刃）、刃 | 「用神之不善而逆用之者也」 | 喜制伏收束，使其受制而有用 |

> 注：原文逆用四神作「煞伤劫刃」，「劫」「刃」指月劫与阳刃（建禄月劫、阳刃格那一路），并非枭神。本库 05 把偏印（枭）也归入逆用之格，是从「枭夺食需制」的实战角度扩充，与原文四神侧重略有出入，断命时以「该神是助命还是攻命」之实质为准，不纠名目。

### 顺逆的现代调用提示

1. **拿到格先问顺逆**：月令取出格神后，先判它属顺用四神还是逆用四神，再定全局该「帮它」还是「治它」。这一步定了整个取用方向，错则满盘皆错。
2. **顺用之神怕受伤、逆用之神怕无制**：顺用之神（如正官）被克被冲即破，逆用之神（如七杀）无制伏之神即为攻身之害。判成败时盯住这一点。
3. **顺逆决定喜忌排序**：顺用格里生扶护卫之神为喜，逆用格里制伏之神为喜。这给 02 定喜忌提供了格局层的依据，两处合看。

## 三、相神：护用之神（论相，子平真诠独有的关键概念）

「相神」是子平真诠最具特色、也最该补进本库 05 的概念。原文：**「月令既得用神，则别位亦必有相，若君之有相，辅者是也。」**沈孝瞻举例（原文）：**「如官逢财生，则官为用，财为相；财旺生官，则财为用，官为相；煞逢食制，则煞为用，食为相。」**总括一句（大意）：凡全局之格全赖某一字而成者，这个字就是相神。

相神之要在于它护着用神成格，相神一旦受伤，格随之破。原文论其轻重：**「伤用神甚于伤身，伤相甚于伤用。」**意即损伤相神比损伤用神还严重，因为相神一倒，用神失护，整个格局垮掉。

| 概念 | 定义 | 在格局中的角色 |
|---|---|---|
| **用神** | 月令所主之格神 | 格之主体，定这是什么格 |
| **相神** | 护用成格、全局赖以成立的那一字 | 格之辅佐，相在则格成，相伤则格破 |
| **忌神** | 伤用神或伤相神之字 | 破格之源（大意：伤用、伤相者皆忌） |

> **这是本库 05 的重点补强处**。05 只说「相神到位则成格」，未点透相神「比用神更要紧、伤相甚于伤用」这层。实战意义：判一个格成不成，找出那个让它成立的关键字（相神），再看这个字有没有被冲、被合、被克。相神安稳，格就稳；相神受伤，纵然用神还在，格也已破。

### 相神的现代调用提示

1. **成格先定相神**：取格后第二问就是「这个格靠哪一字撑着」。如正官无财印护、孤官一个，相神未现，格力薄；得财生官、印护官，相神到位，格力厚。
2. **护相重于护用**：保护命局时，相神所在之柱、所在之字尤其要看它会不会被刑冲合化。相神被合走、被冲掉，是常见的隐性破格，比用神直接受克更易被漏看。
3. **流年大运动相神即动格**：行运引动相神（助之或伤之），往往是格局起落的关键年份，呼应本篇第八节行运。

## 四、成败与救应：因成得败，因败得成（论用神成败救应、论用神因成得败因败得成）

格局有成有败，子平真诠更进一层，讲成败之中还能再翻转。原文：**「八字之中，变化不一，遂分成败。而成败之中，又变化不测，遂有因成得败，因败得成之奇。」**

「因成得败」指本来成格的局，因多出一字反而破了；「因败得成」指本来破格的局，因多出一字反而救回来了。全凭忌神与救应之神的有无。本库 05 口诀「成中有败，必是带忌；败中有成，全凭救应」即本此意（参 05 第二节）。

| 类型 | 含义 | 原文例（照录，趋势化理解） |
|---|---|---|
| **格之成** | 用神得相、配合得宜 | 「化伤为财，格之成也」「印用七煞，格之成也」 |
| **因成得败** | 成格之中又添忌，反破 | 「辛生亥月，透丁为用，卯未会财，乃以党煞，因成得败矣」（伤官本可化作财，却因会成财局去党助七煞而败） |
| **格之败** | 忌神坏用 | 「官印逢伤，格之败也」「煞刃逢食，格之败也」 |
| **因败得成** | 败格之中得救应，反成 | 「煞刃逢食……则食神合官留煞，而官煞不杂，煞刃局清，是因败得成矣」 |

### 成败救应的现代调用提示

1. **断成败要数到最后一字**：不能见用神成格就收，也不能见忌神坏格就断破。要把四柱（连同大运流年）所有字过一遍，看有没有「因成得败」的暗藏忌神，或「因败得成」的暗藏救应。
2. **救应即去忌或合化**：救应之神的作用是制去忌神、或合化忌神、或调停使之不杂。本库 05 称此为「复成」，破而复成者，往往主历险成事、先难后顺。
3. **成败是动态的**：原局成败只是底盘，大运一字进来就可能翻转，断流年务必把运字代入重判一次成败。

## 五、用神配气候得失：格局与调候并参（论用神配气候得失）

子平真诠不只论生克成败，也认调候。原文：**「论命惟以月令用神为主，然亦须配气候而互参之。」**取譬（大意）：英雄生逢其时则事半功倍，遭时不顺纵有奇才也成功不易。

最有名的两组原文断语，正好印证本库 03 调候优先于格局的几处特例：

- **「印绶遇官，此谓官印双全，无人不贵。」**但接着提醒（原文）：**「而冬木逢水，虽透官星，亦难必贵，盖金寒而水益冻，冻水不能生木，其理然也。」**官印本是贵配，遇冬木水旺却因调候失宜而减等。
- **「伤官见官，为祸百端，而金水见之，反为秀气。」**沈孝瞻点破其理（原文）：**「非官之不畏夫伤，而调候为急，权而用之也。」**金水伤官见官之所以反吉，是因冬月调候需火（官），调候之急压过了伤官见官之忌。

> 与本库 03、05 互参：05 列「伤官见官忌，金水木火特例」时未说透原因，本节补出原文之理：特例的根子在调候为急。断金水伤官、冬木等局，调候权重要压过常规格局忌讳，这是子平真诠亲口认的「权而用之」。

## 六、用神格局高低：有情、有力（论用神格局高低）

同样成格，为何有的大贵有的小贵？沈孝瞻给出定层次的总纲。原文：**「八字既有用神，必有格局，有格局必有高低……然其理之大纲，亦在有情、有力无力之间而已。」**配套名句（大意，通行本）：**「格局有高低，财官有厚薄。」**

后世据此整理出「有情、有力、有情而兼有力（最上）、无情、无力」几档。把握两条主轴：

| 主轴 | 高（上格倾向） | 低（减等倾向） |
|---|---|---|
| **有情 / 无情** | 用神与相神生克配合得宜、彼此有情、全局贯通 | 用相之间相战相碍、配置勉强、各不相顾 |
| **有力 / 无力** | 用神相神透干通根、得令得地、力足 | 用神相神虚浮、失令无根、被合被制、力弱 |

> 与本库 05 第四节「定层次三问」对应：05 问「格成否、用神有力否、全局流通否」，正是此处「有情（流通配合）」与「有力（透干通根）」二轴的操作化。最上格是既有情又有力（配合得宜且力足），最次是无情又无力。断高低时两轴各打一分再合判。

## 七、用神变化：月令藏干透与不透（论用神变化）

月令往往藏不止一个十神，透出哪个，格随之变。原文：**「用神既主月令矣，然月令所藏不一，而用神遂有变化。」**沈孝瞻取譬（原文）：**「假使寅月为提，不透甲而透丙，则如府官不临郡，而同知得以作主。此变化之由也。」**意即寅月本气甲木不透、却透了中气丙火，则格随丙火变，如同正印（府官）不到任、由代理者（同知）主事。

理解要点：

1. **透谁取谁，本气优先**：月支藏干以本气为主，本气透干则格定于本气；本气不透而中气、余气透出，则格变于所透之神（参 05 取格步骤）。
2. **变化看会合刑冲**：除了透干，月令藏干还会因地支会合刑冲被引动或被牵制，进而改变格局取向。
3. **一格可兼数神**：藏干多透时，可能正格之外兼出他神，需分主次（谁透得清、谁通根、谁被合），再定以何为主格、何为相神。

## 八、八格顺逆要点（补 05 的子平真诠精到论断）

本库 05 已列八格喜忌速查，本节不重复列举，只补子平真诠对各格最精到的论断与原文口径，供深判时回核。逐格按「顺用 / 逆用」归类。

| 格 | 类别 | 子平真诠论断要点（参 05 速查） |
|---|---|---|
| **正官** | 顺用 | 喜财生、印护，忌伤官见官与刑冲。沈孝瞻视正官为「循理之贵」，宜清不宜杂，最忌伤官填实与官星被合 |
| **财格** | 顺用 | 喜食伤生财、官星护财，身旺能任。忌比劫夺财；身弱财旺则喜印比助身，谓之「财滋弱主」需先顾身 |
| **印格** | 顺用 | 喜官杀生印，忌财星坏印（贪财坏印）。原文论调候时点出「印绶遇官……无人不贵」，但冬木水旺则减等（参本篇五） |
| **食神** | 顺用 | 喜身旺、财星流通（食神生财），忌枭神夺食。食神一字最怕逢枭，谓之「夺食」，是食神格首要忌神 |
| **七杀** | 逆用 | 喜食制、印化、刃合，忌财党杀攻身、杀无制。原文「煞逢食制，则煞为用，食为相」，食神即七杀格的相神 |
| **伤官** | 逆用 | 佩印、生财两路皆可用，忌见官（除金水、木火调候特例）。原文「伤官佩印……用之夏木其秀百倍」「伤官带煞……用之冬金其秀百倍」 |
| **阳刃** | 逆用 | 喜官杀制刃，忌刃无制而冲提纲。阳刃刚旺，得官杀收束方为权贵 |
| **建禄月劫** | 比劫立格 | 月令临官（建禄）、月劫，日主自旺不入八格，须于他位取财、官、煞为用，顺逆视所取之神而定 |

> 用法：本表只是子平真诠论断的索引，每格的完整喜忌、破格救应仍以 05 第二节速查为操作主力，遇疑回本篇前六节核「顺逆 / 相神 / 成败」之理。

## 九、行运：论运与看命无二法（论行运）

子平真诠把大运视作命局的延伸，取法与看命同。原文：**「论运与看命无二法也。」**接着（原文）：**「看命以四柱干支，配月令之喜忌，而取运则又以运之干，配八字之喜忌。」**即把大运每一字代入原局，配原局的喜忌统看，为喜为忌，吉凶自现。

沈孝瞻给出喜运、忌运的判准（原文撮要，趋势化理解）：

- **喜运（原文）**：「命中所喜之神，我得而助之者是也。」举例如「官用印以制伤，而运助印」「伤官佩印，而运行官煞」「阳刃用官，而运助财乡」之类，皆美运。
- **忌运（原文）**：「命中所忌，我逆而施之者是也。」举例如「正官无印，而运行伤」「七煞食制，而运逢枭」「伤官佩印，而运行财」之类，皆败运。

| 行运判读 | 原文口径 | 趋势 |
|---|---|---|
| **运助用神 / 相神** | 我得而助之 | 利于该运引动之事象显发，倾向顺遂 |
| **运伤用神 / 相神** | 我逆而施之 | 易主该运阻滞、破格之事，需注意 |
| **运补救应 / 去忌神** | 败而得救 | 原局有病者逢此运易转好，先难后顺 |

### 行运的现代调用提示

1. **运字代入再判成败**：每行一运，把运干支当作多出来的一字，代回原局重判一次顺逆、相神、成败（参本篇二、三、四）。这与本库《三命通会》行运篇「运之吉凶取决于运对原局用神的作用」完全一致。
2. **运动用神相神则吉凶显**：大运引动（生扶或克伤）用神、相神之年，是格局起落的关键节点；不动用神相神的运，吉凶平淡。
3. **运与流年合参**：大运定十年大势，流年定一年休咎，两者代入原局合看，呼应本库行运与流年分析，不可只看运不看年。

## 使用原则

1. **先取格、再顺逆、再相神、后成败**：拿到命局先以月令取格（参 05），判格神属顺用还是逆用，找出护格的相神，最后把四柱连同大运过一遍数清成败救应。这是子平真诠的标准流程，环环相扣，不可跳步。
2. **分清两种「用神」**：沈孝瞻的「用神」是月令取格之神，本库 02 的「用神」是扶抑调节之神，名同实异。引用本篇时务必标明指的是哪一种，避免混淆。
3. **相神是本库 05 的补强重点**：判格成败盯住相神，记住「伤相甚于伤用」，相神被刑冲合化是最易漏看的隐性破格。
4. **调候可压格局常忌**：遇金水伤官、冬木等局，调候为急时可「权而用之」，常规格局忌讳让位于调候（参本篇五、本库 03）。
5. **断语趋势化**：用「倾向 / 利于 / 易 / 需注意」，原文虽有「无人不贵」「为祸百端」等重断之语，引用时降级为趋势提示，不承诺祸福。

- 篇目归属（沈孝瞻原著章次）：用神总纲、顺逆、相神、成败救应分见「论用神」「论用神成败救应」「论用神因成得败因败得成」诸章；配气候出「论用神配气候得失」；格局高低出「论用神格局高低」；用神变化出「论用神变化」；行运出「论行运」。徐乐吾评注随各章附后，本篇取沈氏正文，未引徐注。
- **存疑与取舍**：
  - 逆用四神的用字，经 ctext、太极书馆、东里书斋三源一致核为**「煞伤劫刃」**（劫指月劫、刃指阳刃），照原文录入。坊间另有作「杀伤枭刃」者，与三种公版底本不合，本篇不采。本库 05 将枭（偏印）一并归入逆用之格，是实战角度的扩充而非原文原话，已在第二节注明出入。
  - 相神轻重原文为**「伤用神甚于伤身，伤相甚于伤用」**，照录。坊间转述作「伤用神为忌神，伤相神为忌神」者属撮意，义近而非原句，本篇取原文，撮意处已标「大意」。
  - 行运章首句原文为**「论运与看命无二法也」**，照录。「论行运即同看命」一语为后人撮意标题式转述，非逐字原文，本篇用原句。
  - 格局高低原文落点在**「有情、有力无力之间」**，照录；后世整理的「有情 / 有力 / 无情 / 无力」四分法属归纳框架，便于操作，已标明为后世整理。
  - 「当顺而顺，当逆而逆，配合得宜，皆为贵格」「格局有高低，财官有厚薄」二句通行本广见，文字基本一致，按通行本录，标「大意 / 通行本」以示审慎。
  - 成败救应所引命例句（辛生亥月、官印逢伤等）照录原文用字，其断语一律作趋势理解，不作铁口。
