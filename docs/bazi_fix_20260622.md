# 八字排盘报错修复

## 时间
2026-06-22 16:50 (Asia/Shanghai)

## 问题
用户反馈「明白中」八字排盘功能报错（显示"计算遇到问题"）。

## 根因分析
`computeBazi()` 函数的 try 块内定义了局部变量 `year2/month2/day2`（真太阳时校正后的日期），但以下两个函数内部错误引用了这些变量：

1. **`getBaziCalcData()` 第12123行**：`computeDayun(pillars, sex, year2, month2, day2, ...)` 
   - `getBaziCalcData` 是独立函数，参数为 `(year, month, day, hour, sex)`
   - `year2/month2/day2` 在其作用域内未定义 → ReferenceError → 触发 catch 显示报错页

2. **`renderNewBaziModules()` 第12903行**：`renderLiuNianOverview({ ..., birthYear: year2 })`
   - 同理，`renderNewBaziModules` 是独立函数，`year2` 未定义

## 修复
| 位置 | 修改前 | 修改后 |
|------|--------|--------|
| `getBaziCalcData` L12123 | `computeDayun(..., year2, month2, day2, ...)` | `computeDayun(..., year, month, day, ...)` |
| `renderNewBaziModules` L12903 | `birthYear: year2` | `birthYear: year` |
