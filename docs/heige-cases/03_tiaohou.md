# 命例三 · 调候为急（乙木生子月，水寒木浮无火，调候优先于扶抑）

> 教学原型：日主生于严寒之月，满盘水寒、独缺火暖，气候偏枯。此时取用神，**调候的优先级压过扶抑**。演示「冬生水寒无火」的命，为何纵然印旺身不弱，第一用神仍是火，呼应 references/03 调候为先的原则。

- **演示原型**：调候为急。日主乙木生于子水月，水正当令、壬水双透、满盘印星，木得水生看似不弱，实则**水多木浮、天寒地冻**。全局无火暖局，寒凝不发。调候用神丙火，优先于一切扶抑。
- **虚拟生辰**：公历 1972 年 12 月 30 日 23 时 30 分，男，按北京时间未做经度校正。

## 三、读盘定盘面

- **日主**：乙木（阴木），花草藤蔓之木，柔韧、善生长、能屈能伸。
- **月令**：子水（水），是乙木的印星（偏印）当令。时值冬至后、寒气最盛之际。
- **天干**：年、月两壬水（正印）夹生日主，时干戊土（正财）。天干是「双印生身、一财微露」。
- **关键合冲**：地支三子与日支未，三组子未六害；全局**子水三见、无一火星**，日支未中虽藏丁火、己土、乙木，是日主仅有的根与火源，却被三子相害冲克。盘面是「满局寒水、孤未含微火被围」的格局。

## 四、旺衰判定

按五看逐条推：

1. **月令**：生于子月，乙木在子为「病」，但子水是生木之印、当令而旺，**水来生木，日主得印之生而不弱**（水多木相）。〔依据：子月水印当令生木〕
2. **通根**：日主乙木坐下未土，未中藏乙木（本气余气），是乙木的微根；又未藏丁火、己土。这是日主唯一的根，却被三子六害所冲。〔依据：日坐未藏乙为微根、遭三子害〕
3. **生扶**：印星水力量高达 6.0（三子＋双壬），生身之力极盛；比劫木仅 1.2。**生身的全是印水，比劫极少**。〔依据：水 6.0、木 1.2，同党靠印〕
4. **党众**：同党（印比）7.2 对异党（食伤财官）2.5，同党占比 74%，印星一柱独大。〔依据：脚本同党 74%、水 6.0〕
5. **定档**：从党众与力量看，日主由旺印生扶，量化偏旺；但**这是「印重」之旺，旺得虚而寒**。真正的盘面性质是「水多木浮、寒木不荣」，旺衰档位之外，更要紧的是气候问题。〔依据：印重水寒 + 木浮〕

> 关键一步：本盘**不能简单按身旺取财官食伤泄耗**。原因有二。其一，旺在印水，水多反而泡根、漂木（水泛木浮），印多成病，单纯泄身于财（戊土）会引动土水相战、且戊土难挡三水；其二，更根本的是**气候**：子月严寒、满盘冰水、全局无火，寒木僵冻不能生发。此时必须把「调候」提到取用神的第一位。〔依据：references/02 误区三·忽略调候；水多木浮印多为病〕

## 五、用神喜忌

**调候优先**是这盘的命脉，逐层推：

- **调候（第一优先）**：乙木生子月，《穷通宝鉴》明示「乙木子月，专用丙火解冻」，寒谷回春全凭丙火暖局。本盘满水无火、寒凝至极，**丙丁火既能暖局解冻，又能泄旺木与化旺水之气（水生木、木生火，火来流通泄秀），还能制约水泛**，一字三用，是全局第一急需。〔依据：references/03 乙木子月专用丙火解冻；调候 ≥ 扶抑〕
- **扶抑（次位）**：日主由旺印生扶、量化偏旺，本不需再补印比；真正该做的是**泄旺木之秀、并疏导过旺之印水**。火（食伤）泄秀暖局、土（财）培根止水（戊土制水、又为乙木立足之实地），方向与调候同辙。〔依据：references/02 印重宜泄宜耗、戊土止水培根〕
- 调候要的火、与疏导印水所需的火土同向，用神层次清楚：**火为第一，土为佐**。最忌再添水（印泛）与金（生水克木）。

定论：

| 类别 | 五行 / 十神 | 理由 |
|---|---|---|
| **用神** | 火（食神、伤官） | 丙丁火解冻暖局、泄旺木与寒水之秀、令满局寒水有去处，是寒木回春的命门。原局火仅 0.5（藏未中丁火一点），全靠运补。〔依据：子月调候专用火 + 火泄秀通关〕 |
| **喜神** | 土（正财、偏财） | 戊己土培木之根、堤防止住泛水（土克水），又是火的去处（火生土）。戊土正财已透时干，是局中难得的实地。〔依据：土止水培根 + 财泄火〕 |
| **忌神** | 水（正印、偏印） | 印水已泛滥成灾（水 6.0），再添水则木愈浮、局愈寒。头号忌讳。〔依据：水多木浮、印多为病〕 |
| **仇神** | 金（正官、七杀） | 金生水助泛、又克日主乙木，助纣为虐。〔依据：金生忌水又克身〕 |
| **闲神 / 半喜** | 木（比劫） | 比劫帮身本中性，但木多可助火（木生火）、亦能分泄印水之生，于回春略有助，按半喜看；唯忌纯木无火仍寒。〔依据：木生火助暖、但无火仍寒〕 |

一句话：这盘最该补的是**火（暖局解冻）**，最得力的是**土（止水培根）**，最怕再见**水（印）金（官）**。这正是「调候压过扶抑」的活例：量化身旺，取用却不走克泄财官的常路，而是先要一把火。

## 六、格局成败

- **取格**：月令子，本气癸水（偏印）未透天干，但年月双壬（正印）透出，**印格（偏印格 / 印重格）**成立。〔依据：references/05 月令印星、双印透干〕
- **成败**：印格喜官杀生印、忌财坏印；但本盘是**印旺成灾**的变局，常规「印格喜官生」反而不适用（再生印则水更泛）。此局的成败不看官印相生，而看**调候之火是否到位**：得火暖局泄秀，则寒木回春、印水有去处，印格转清而成；无火，则满盘寒冻、印多身滞，格局虚寒难发。时干戊土正财能制水培根，是局中一点救应（财损印、止泛水），但孤土三水，力薄需运助。〔依据：印重需火调候转清 + 戊土制水救应〕
- **定层次**：印星太旺、调候之火原局全无，是**偏枯待补**之局。格局清浊全系于大运能否补上火土。神煞虽贵（天乙、月德、将星、华盖齐聚，主聪明清贵、有文气与贵人），但**调候不及则纵有贵气亦减层次**，属**底子清贵、待运回暖方发**的中等格局。〔依据：references/03 调候不及减层次；references/05 偏枯减等〕

## 七、大运流年要点

大运顺排，1 岁起。原则：行**火（食伤）土（财）**之运为顺（补调候、止泛水），行**水（印）金（官）**之运为逆（助寒助泛）。这盘的人生主线，就是「等火来」。

- **癸丑（3-12 岁）**：癸水偏印（忌）、丑为湿土藏水藏金，**寒湿加重**。幼年体弱或家境清寒之象，水印过旺、读书虽有靠山而局面偏冷。〔依据：癸水印忌 + 丑湿土〕
- **甲寅（13-22 岁）**：甲木劫财、寅木藏丙火（用神之苗）。**寅中丙火给满盘寒水透进第一丝暖意、寅木又生火**，比劫帮身、木气转旺。少年到青年，身心渐开、读书与志气见长，是寒局里第一缕回暖。〔依据：寅藏丙火暖局 + 木生火〕
- **乙卯（23-32 岁）**：乙卯纯木比劫（半喜），木气极旺。**木多能引火、分泄印水之生**，但**纯木无火则暖意仍不足**，主自立、奔走、积累人脉与本事，局面活而未大发。〔依据：木旺引火分印、无火暖意不足〕
- **丙辰（33-42 岁）**：天干丙火伤官透（用神到位），地支辰为湿土藏水、又能蓄水培木。**调候之火终于透出天干，寒局见太阳**，这是全局的关键转暖：才华得以发挥、寒木开花，事业与名声起步真正成形。唯辰中带水、湿土晦火，火力尚需巩固。〔依据：丙火用神透 + 辰土培木〕
- **丁巳（43-52 岁）**：天干丁火食神透（用神），地支**巳火当旺、巳中藏丙戊庚**，火土金俱来。**火暖到位、巳火大补调候，又生戊土财**。这是这盘真正的黄金段：寒木彻底回春、食神生财、暖局通明，事业财运与心境一起转旺。〔依据：巳火大补调候 + 食神生财〕
- **戊午（53-62 岁）**：天干戊土正财透（喜，止水培根），地支**午火当旺（食神禄地）**，火土双美。**调候与财源同到、午火暖局戊土实地**，是晚运得财、心安身暖的好运，财旺而身有去处。唯午与原局未合、与子水暗冲，主有变动奔波。〔依据：午火暖局 + 戊土财得地〕

**关键流年**（落在当前 2024 起、戊午火土用神运的背景下，正是回暖发力之时）：

- 2025 乙巳、2026 丙午、2027 丁未：火气连年到位（巳午未火方），**调候大补、暖局通明的几年**，是发挥才华、成事进财、心境舒展的好窗口，宜主动出击。〔依据：巳午未火补调候用神〕

## 八、分维度断语

**性格**：乙木为花草藤蔓之木，**柔韧、善应变、能屈能伸、心思细腻、有艺术与生发之性**〔依据：日主乙木〕。印星（水）极旺，**好学、悟性高、心思深、念旧、依赖直觉与靠山**〔依据：印水旺〕。配天乙、月德、华盖、将星，**清贵、有文气、悟性与灵性突出**〔依据：贵人华盖齐聚〕。短板在「水寒木浮」：早年易**偏内向、思虑过重、行动力被寒气压住、缺一把火点燃热情与决断**〔依据：水寒无火、木浮〕。火运一到，人就开朗、有冲劲。修养上宜主动求暖（向阳、热情、行动），别困在思虑里。

**事业**：这盘事业的总开关是「火」，**得火则发，无火则滞**。早年（火未到）宜积累、读书、练本事，不急于求成；**33 岁丙辰、43 岁丁巳火运起，才是事业真正成形与起飞期**〔依据：丙辰丁巳火运补调候〕。方向走食伤生财一路：印旺华盖加身，利**文化、创作、设计、研究、教育、策划、咨询**等靠才华与悟性吃饭的领域〔依据：印旺华盖 + 食伤为用〕；火为用，亦利**与温暖、光明、文创、传播相关**的行业。行业按用神取火土：**文创传媒、教育文化、餐饮温暖类、能源光电、地产实业（土）**为宜，避开纯水（水产、远洋、寒湿环境）与纯金（冷硬五金）。方向利**南方（火）、中部（土）**。〔依据：用神火土〕

**财运**：财星为戊己土，**原局戊土正财透时干、有一点实地，但被三水围、力薄**，主**财源要靠自己一点点筑实、且需火来生财（食伤生财）**〔依据：戊财透而被水围〕。得财之道在**先暖局、再以才华生财**：火运里食神伤官生财最旺（丙辰、丁巳、戊午及巳午未年），是进财的主轴。早年水旺无火，财难聚、易清寒；中年火到，财随才华而来。理财宜**务实置产（土为喜，实地实业能止泛水）**，避寒湿投机与水类风险。〔依据：食伤生财 + 土为喜〕

**婚姻感情**：男命以财为妻、日支为夫妻宫。本盘**财星（戊己土）偏弱被水围、日支未又遭三子六害**，夫妻宫不宁，主**感情早年易聚散不定、配偶或缘分来得偏迟或需磨合**〔依据：财弱被围 + 日支未三害〕。桃花三见（子为桃花、三子皆是），**异性缘旺、情思丰富**，但桃花在印位、多而虚，易陷思虑型情感〔依据：桃花三见在印〕。宜**晚婚**（火暖运到、心定了再成家更稳）；找**温暖开朗、能给热度、属火或属土（实地）的另一半**，能暖局、能让其落地。逢动夫妻宫之年（子年、丑未冲合之年）多沟通经营。〔依据：调候用火 + 日支被害〕

**健康**：五行配脏腑，本盘**水过旺、火全缺、木浮**。第一隐患在**肾、泌尿、生殖与寒湿之疾**（水泛成灾，易水肿、寒湿、泌尿与肾系统、循环偏弱、手脚不温）〔依据：水过旺主肾、寒湿〕；第二隐患在**心血与小肠、眼目、情志**（火枯，阳气不振，易心阳不足、情绪低、睡眠浅）〔依据：火枯主心阳〕；木浮则**肝胆、筋、神经**亦需调养。高发在寒水旺的运与年，**尤其早年癸丑运与亥子丑水方流年**，宜保暖、温补、动则生阳、忌寒湿生冷与水边久居。火运到（中年后）健康随暖局好转。〔依据：水旺火枯之运年〕

**学业才华**：印星（水）极旺、当令，又天乙月德华盖将星齐聚，**主聪明清贵、记性悟性俱佳、有文采灵性、读书与靠山缘厚**〔依据：印旺当令 + 贵人华盖〕。这是典型的**印重文气命**：利走读书、研究、文化、艺术、靠悟性与积累立身的路。功课在「印多则思虑重、行动迟」，需**配火（食伤）把所学吐出来、化为作品与成果**，否则易满腹才学而难落地。〔依据：印旺华盖 + 需食伤泄秀〕

**六亲**：父母与出身，印星（水）为母、旺极当令，**母缘深、靠山足、得长辈与师长庇荫，但早年家境或偏清寒、偏冷**〔依据：印旺为母、水寒〕。父亲方面，财（土）为父，财弱被围，父或操劳、或缘分平淡。兄弟方面，比劫（木）仅一点，**手足不多、各自独立**，木运（甲寅乙卯）来时同辈助力增。配偶方面，妻星弱而夫妻宫被害，重在晚成与磨合经营。子女方面，男命以官杀（金）为子，原局金缺、官杀无（仅运补），**子女缘着落在中晚年金运（庚申辛酉），来得偏迟**，相处宜顺其自然。〔依据：官杀金缺待运〕

## 九、调候趋避

- **方位**：宜**南方（火）、中部（土）**发展定居，南方暖地尤利；忌久居正北（水）寒湿之地与水边。〔依据：用神火土〕
- **颜色 / 数字**：宜红紫橙（火）、黄棕（土）；数字宜 2、7（火）、5、0（土）；忌黑蓝（水）、白银（金）。〔依据：用神色〕
- **行业**：宜文创传媒、教育文化、餐饮温暖类、能源光电、地产实业；忌纯水（水产远洋、寒湿环境）与纯金（冷硬五金）。〔依据：用神火土〕
- **养生**：保暖温补、动则生阳、多晒太阳、忌寒湿生冷与久居水边，护肾与心阳。〔依据：水旺火枯〕
- **关键转折**：**33 岁丙辰、43 岁丁巳火运**是这盘由寒转暖、事业财运起飞的黄金段，宜趁火暖之运全力发力。〔依据：丙辰丁巳补调候〕

## 十、综合总评

乙木生子月，双壬透干、满盘寒水、独缺一火，是一棵**生在隆冬、被冰水泡着根、等一缕阳光的花草之木**：柔韧、清贵、聪明有文气（印旺华盖、贵人满局），却被寒气压住了生发之力。这盘的命门一字概括就是**火**：调候之火压过一切扶抑，得火则寒木回春、才华吐秀、印水有去处；无火则满盘冰冷、思虑成滞、清寒难发。它最好地说明了一个道理：量化身旺不等于该用克泄财官，**冬生水寒之命，第一用神永远先看那把火**。一生大势是**先寒后暖、靠运回春**：早年水旺清寒、读书积累，13 岁寅木透火见第一缕暖，**33 到 62 丙辰丁巳戊午这三十年火土连珠才真正回暖大发**，晚年再入水方需保暖守成。趋吉八字：**近火土、向南中，求温暖、化才学，晚成事、暖立身。**
