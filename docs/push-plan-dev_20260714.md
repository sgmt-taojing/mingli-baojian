## 功能清单

### 推送内容分级
| 等级 | 代码 | 推送内容 |
|------|------|----------|
| 缘主（免费） | free | 每日宜忌基础版（3个吉时）+ 引流卡片 |
| 年度会员 | annual | 每日完整四板块 + 月运 + 节气 + 年度运程 + 化解方案 |
| 终身会员 | lifetime | 年度全部 + 家庭运程 + 专属化煞 + 跨年衔接 |

### 推送时间矩阵
- **每日 07:30**：每日宜忌+吉时（所有用户）
- **每日 18:00**：晚间提醒（会员）
- **节气前1天**：24节气养生+运势（会员）
- **传统节日**：8大节日专题推送（所有用户，含引流）
- **每月1号 08:00**：月运报告（会员）
- **跨年12月-1月**：9个关键节点推送（12-15→立春当日）

### 跨年推送时间线（9个节点）
1. 12-15：年度运势回顾报告
2. 12-22（冬至）：冬至养生专题 + 新年化解方案预告
3. 12-28：新年化煞物品准备清单（终身会员）
4. 12-31：跨年祝福 + 流年交接提醒
5. 01-01：新年第一推 + 本年度流年概览
6. 01-05（小寒）：年度化解方案启动指南
7. 01-20（大寒）：立春前准备 + 个性化化煞时辰
8. 立春前3天：流年切换最终准备 + 物品清单
9. 立春当日：流年正式切换 + 完整年度化解方案

### 引流转化设计
- 7种升级解锁卡片（吉时限制、财运方位、化解方案等）
- 会员到期前7天续费提醒
- 节气/节日推送中嵌入3种会员预告
- 5种引流文案风格（温和、好奇、价值、紧迫、社交）
