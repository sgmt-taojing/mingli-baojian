# 系统优化记录 (2026-06-14 11:45)

## 本次修复的问题

### 1. 首页"更多"菜单无法点击
**问题原因**：PC端CSS中设置了 `.more-menu { display: none !important; }`，完全隐藏了下拉菜单

**解决方案**：
- 移除 `display: none`
- 改用 `transform` 和 `opacity` 控制显隐
- PC端改为hover触发，移动端保持点击触发
- 添加 `pointer-events` 控制交互状态

### 2. 改名/姓名分析无法出结果
**问题原因**：
- 康熙笔画字典不够完整（原660字）
- `getKangxiStroke` 函数遇到未知字返回0
- `calculateWuge` 遇到笔画为0直接返回null，没有错误提示

**解决方案**：
- 扩充康熙笔画字典至3000+字
- 添加繁简对照表
- 增强 `getKangxiStroke` 函数：
  - 先查字典
  - 再查繁简对照
  - 返回0时提供友好提示
- 改进 `calculateWuge` 函数：
  - 检测未知字并提示
  - 返回详细错误信息
  - 不再静默失败
