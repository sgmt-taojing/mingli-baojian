# 奇门遁甲 V3 校准引擎

**日期**: 2026-07-14  
**状态**: 已完成  
**引擎**: QimenV3

## 概述

为奇门遁甲模块开发 V3 校准引擎，基于天文算法精确计算节气，校准排盘逻辑，并集成到 hub.html。

## 校准典籍

- 《神奇宝海》
- 《奇门遁甲秘笈大全》
- 《钦定授时通考》

## V3 校准要点

### 1. 天文节气定局
- 使用 VSOP87 简化算法计算太阳黄经，精度 ±0.01°
- 冬至（黄经270°）后用阳遁，夏至（黄经90°）后用阴遁
- 每个节气分上中下三元，每元5天，以日干支甲己日为符头
- 摒弃公历近似日期法，改用天文算法精确判断节气交接日

### 2. 六仪三奇排布
- 阳遁：戊己庚辛壬癸丁丙乙（顺布六仪，逆布三奇）
- 阴遁：戊己庚辛壬癸丁丙乙（逆布六仪，顺布三奇）
- 按九宫数序飞布，戊起于局数对应宫位

### 3. 天盘转动
- 值符随时辰旬首所在宫位
- 天盘飞布：阳遁顺飞，阴遁逆飞
- 以时辰地支宫为目标进行旋转

### 4. 八门校准
- 值使从旬首宫起，随时辰递宫
- 原始宫位：休1 生8 伤3 杜4 景9 死2 惊7 开6
- 中宫无门，跳过

### 5. 九星校准
- 天蓬(冬至)→天任→天冲→天辅→天英→天芮→天柱→天心→天禽
- 值符星 = 旬首所在宫的原始星
- 九宫飞布含中宫5

### 6. 八神校准
- 值符→螣蛇→太阴→六合→白虎→玄武→九地→九天
- 值符跟值符星所在宫
- 中宫无神，跳过

### 7. 格局判断
- **吉格**: 青龙返首、飞鸟跌穴、玉女守门、真诈格、假诈格、重诈格、五假格（天假/地假/人假/神假/鬼假）
- **凶格**: 白虎猖狂、腾蛇妖娇、太白入荧、荧入太白、青龙折足、朱雀投江、伏吟、反吟
- **平格**: 伏吟

### 8. 五不遇时
- 时干克日干（七杀关系），且阳克阳/阴克阴
- 甲日庚午时，乙日辛巳时等十组

## 集成方式

1. QimenV3 代码合并到 `engine-v3-bundle.js`（IIFE 隔离作用域）
2. `hub.html` 中 `runQimenEngine()` 函数在化解面板之后追加 V3 校准面板
3. V3 面板包含：
   - 遁局信息（天文节气 + 太阳黄经度数）
   - 九宫天地盘（3×3 网格，后天八卦排列）
   - 八门、九星、八神分布
   - 空亡/马星标注
   - 格局判断
   - V3 断语

## 导出 API

```javascript
window.QimenV3 = {
  // 核心排盘
  qimenCalcV3(year, month, day, hour, juType?) → 完整排盘
  getDunJu(year, month, day, hour, juType) → 定遁局
  getJieqiPrecise(year, month, day) → 精确节气
  arrangeDiPan(dun, ju) → 地盘六仪三奇
  arrangeTianPan(dipan, hourGzIdx, ju, dun) → 天盘转动
  arrangeMen(dun, ju, hourGzIdx, dipan) → 八门
  arrangeStars(dun, ju, hourGzIdx, tianpan) → 九星
  arrangeShen(dun, hourGzIdx, tianpan) → 八神
  // 分析
  getGeju(panData) → 格局判断
  getWuBuYu(dayGanIdx, hourGzIdx) → 五不遇时
  analyzeQimenFull(panData) → 完整分析
  // 常量
  JIEQI_JU_TABLE, BAGUA, JIU_GONG, STEMS, BRANCHES, ...
};
```
