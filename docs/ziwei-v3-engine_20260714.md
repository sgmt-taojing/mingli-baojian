# 紫微斗数 V3 校准引擎文档

## 概述

ZiweiV3 是紫微斗数排盘引擎的 V3 校准版本，集成在 `app/js/engine-v3-bundle.js` 末尾（引擎 7/7）。

## 参考经典

- 《紫微斗数全书》— 命宫定位、安星法、四化、格局
- 《太微赋》— 格局论断、星曜组合
- 《紫微斗数精成》— 定局、安紫微、辅星煞星

## V3 校准要点

### 1. 命宫定位
- 寅起正月，顺数月，逆数时 → 命宫
- 身宫：寅起正月，顺数月，顺数时

### 2. 定局
- 命宫天干 → 纳音五行 → 局数
- 水二局、木三局、金四局、土五局、火六局

### 3. 安紫微
- 生日÷局数，查表定紫微宫位
- 使用《紫微斗数精成》安紫微表

### 4. 主星安星
- **紫微系（逆行）**：紫微→天机(隔1)→太阳(隔2)→武曲(隔1)→天同(隔1)→廉贞(隔3)
- **天府系（顺行）**：天府→太阴→贪狼→巨门→天相→天梁→七杀→破军(隔3)
- 天府与紫微以寅申线对称：`天府 = (4 - ziweiPos + 12) % 12`

### 6. 辅星
- **左辅**：正月从辰起顺行
- **右弼**：正月从戌起逆行
- **文昌**：子时从戌起逆行
- **文曲**：子时从辰起顺行
- **天魁/天钺**：按日干查（昼贵/夜贵）

### 7. 煞星
- **擎羊/陀罗**：年干禄前/后一位
- **火星/铃星**：年支三合局起寅
- **地空/地劫**：子时从亥起，空逆劫顺
- **天马**：按年支三合局
- **天刑**：按日干查
- **红鸾/天喜**：红鸾从卯起逆数年支，天喜为对宫
- **解神**：按月查

### 8. 大运
- 从命宫起，阳男阴女顺行，阴男阳女逆行
- 每宫10年
- 起运年龄 = 局数

## API

### 核心排盘
- `ziweiCalcV3(year, month, day, hour, sex)` — 完整排盘
- `getMingGong(month, hour)` — 命宫定位
- `getShenGong(month, hour)` — 身宫定位
- `getJu(ganIdx, zhiIdx)` — 定五行局
- `anZiwei(ju, day)` — 安紫微
- `anStars(ziweiPos)` — 安主星
- `getSihua(yearGan)` — 四化
- `getAuxStars(month, hourIdx, dayGanIdx)` — 辅星
- `getShaStars(yearGanIdx, yearZhiIdx, hourIdx, dayGanIdx, month)` — 煞星
- `getDayun(mingIdx, ju, yearGan, sex, birthYear)` — 大运

### 分析
- `analyzeSanfang(panData)` — 三方四正分析
- `getGeju(panData)` — 格局判断
- `analyzeZiweiFull(panData)` — 完整分析

### 常量
- `ZW_GONGS` — 十二宫名称
- `ZW_STARS` — 十四主星
- `SIHUA_TABLE` — 年干四化表
- `JU_MAP` / `JU_REVERSE` — 局数映射
- `NAYIN_WUXING` — 纳音五行表
- `ZIWEI_POS_TABLE` — 安紫微查表
- `STAR_STRENGTH` — 庙旺平陷表

## 集成位置

- **引擎代码**：`app/js/engine-v3-bundle.js` 末尾（IIFE #7）
- **HTML集成**：`app/divination-hub.html` → `runZiweiEngine()` 函数
- **V3面板**：在化解方案之后、`_showEngineResult` 之前渲染
- **面板内容**：对比表 + 四化落宫 + 三方四正 + 经典格局 + 大运排布 + V3概述/建议
