## 实现功能

### 1. 中央主图（Canvas绘制）
- **五行生克图**: 五角形布局，展示金木水火土
  - 相生关系：绿色实线箭头（金→水→木→火→土→金）
  - 相克关系：红色虚线箭头（金→木→土→水→火→金）
  - 中央太极图：阴阳鱼图案
  
- **八卦环**（半径150px）：乾坎艮震巽离坤兑
  - Unicode符号显示（☰☱☲☳☴☵☶☷）
  - 点击显示卦象详细解读

- **天干环**（半径200px）：甲乙丙丁戊己庚辛壬癸
  - 点击显示天干属性与喜忌

- **地支环**（半径250px）：子丑寅卯辰巳午未申酉戌亥
  - 点击显示地支与生肖对应关系

### 2. 奇门遁甲九宫格（独立Canvas）
- 3×3宫格布局（600×600px）
- 每个宫位显示：
  - 宫位编号（1-9）
  - 八门（休生伤杜景死惊开）
  - 九星（天蓬、天芮、天冲、天辅、天禽、天心、天柱、天任、天英）
  - 八神（值符、螣蛇、太阴、六合、白虎、玄武、九地、九天）
  - 吉凶判断（吉/凶/半吉/大吉）
- 点击宫位显示详细解读

### 3. 右侧解读面板
- 默认显示："点击左侧任意元素查看详细解读与化解建议"
- 点击元素后动态显示：
  - 元素基本信息（颜色、方位、季节、五行等）
  - 特性描述
  - 一句话总结
  - 化解建议/开运建议（3-5条）

### 4. 交互效果
- **Hover效果**: 鼠标悬停时元素放大并高亮
- **点击高亮**: 被点击元素金色边框+阴影效果
- **面板切换**: 点击不同元素时解读面板平滑切换（fadeIn动画）

## 技术实现

### 架构选择
- **单一Canvas方案**: 主图使用单一Canvas绘制所有元素（五行+三环）
- **优势**: 
  - 统一的坐标系统
  - 简洁的点击检测（鼠标坐标计算）
  - 流畅的hover效果重绘
