## 三、新增需求规划

### 3.1 十二生肖模块（优先级：⭐⭐⭐⭐⭐）

#### 3.1.1 本命佛
每个生肖对应一尊本命佛，提供：
- 本命佛介绍（形象、寓意、功德）
- 供奉方法
- 佩戴禁忌

**数据结构：**
```javascript
{
  zodiac: '鼠',
  buddha: '千手观音菩萨',
  image: 'images/qianshou-guanyin.jpg',
  intro: '千手观音是大慈大悲的象征...',
  worship: '每日清晨焚香三拜，持诵圣号...',
  taboos: ['不可佩戴入厕', '不可置于低于腰部位置']
}
```

#### 3.1.2 年度化解建议
按生肖+年份给出：
- 流年运势概述
- 吉凶方位
- 化解方法（开运物品、颜色、数字）
- 忌讳事项

#### 3.1.3 推荐道观/寺庙
- 按生肖推荐对应的道观/寺庙
- 地址、开放时间、门票信息
- 核心景点参观指南（如为景点）
- 最佳参拜时间（节气、吉日）

#### 3.1.4 吉祥物推荐（每年更新）

**按位置分类：**
| 位置 | 吉祥物 | 材质 | 摆放位置 |
|------|--------|------|----------|
| 家里 | 貔貅/金蟾/葫芦/麒麟 | 铜/玉/水晶 | 客厅财位/玄关/书房 |
| 车里 | 平安符/八卦镜/貔貅 | 铜/木/玉 | 仪表盘/后视镜 |
| 身上 | 本命佛吊坠/手串/护身符 | 玉/金/银/檀木 | 佩戴方法 |

**数据结构：**
```javascript
{
  year: 2025,
  zodiac: '鼠',
  home: {
    items: ['貔貅', '金蟾'],
    materials: { '貔貅': ['铜', '玉'], '金蟾': ['铜', '树脂'] },
    placement: { '貔貅': '客厅财位（进门对角线）', '金蟾': '门口朝内' }
  },
  car: {
    items: ['平安符', '小貔貅'],
    materials: { '平安符': ['红布', '黄布'] },
    placement: { '平安符': '后视镜悬挂' }
  },
  body: {
    items: ['千手观音吊坠', '黑曜石手串'],
    materials: { '千手观音吊坠': ['玉', '金', '银'] },
    wearing: { '吊坠': '贴身佩戴，不可外露' }
  }
}
```

## 五、优化原则

1. **只增不减**：不删除现有功能模块
2. **真实可用**：每个按钮点击有反馈、有输出
