# 八字排盘节气精度修复 — 2026-06-22

## 目标
修复乾元命理宝鉴(divination-hub.html)八字排盘的年柱立春分界和月柱节气分界问题，使其与 HeiGe-SuanMing (lunar_python) 引擎对齐。

## 修复内容

### 1. 年柱立春分界
- **问题**：原代码直接用公历年份计算年柱，未考虑立春分界
- **修复**：新增 `getJieDate()` 函数 + `JIE_DATES` 压缩查找表（12节气×150年），立春前属上一年
- **精度**：日级精度，立春当天上午/下午用 `getYearStemBranchExact()` 区分（12:00前旧年，12:00后新年）

### 2. 月柱节气分界
- **问题**：原代码用 `(month+9)%12` 计算月支，未考虑节气
- **修复**：新增 `getMonthBranchByJieqi()` 函数，遍历12节气找到最后一个已过的节，确定月支
- **特殊处理**：1月小寒前属上年子月（大雪后）；12月大雪前属亥月

### 3. 节气日期查找表
- `JIE_DATES` 对象：12个节气，每个含 `base:[月,日]` 和 `offsets` 字符串
- 覆盖年份：1900-2049
- 偏移编码：`0`=基准日, `1`+1天, `2`+2天, `a`-1天, `B`-2天
