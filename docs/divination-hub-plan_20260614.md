## 数据库架构（新增 · 2026-06-14）

### 技术方案
- **数据库**：SQLite（Python内置，无需安装额外依赖）
- **优势**：单文件、无服务器、单核、无网络依赖、易备份
- **路径**：`~/.qclaw/workspace/knowledge-db/knowledge.db`
- **API服务**：`knowledge-api.py`（端口 8891，HTTP REST接口）

### 数据库结构
| 表名 | 用途 | 状态 |

| knowledge_base | 10大领域知识（八字/六爻/风水等） | ⬜ 持续填充中 |
| faith_knowledge | 信众知识（神仙/经文/口诀/禁忌/参拜/吉日/健身） | ⬜ 框架待建 |

### API端点
