# 命例库 · cases

> 八字命理 skill 的**实战回测层**。方法论（SKILL.md）与断语规则（references/）告诉你「该怎么算」，命例库告诉你「一个具体命盘，从排盘到结论，整条链是怎么走完的」，像技术分析的历史回放。

## 一、这个目录是做什么的

`references/` 是分层的方法论与断语字典，`SKILL.md` 是 0-10 步的推演流程。但单看规则，容易停在「正确而抽象」的层面：知道身强用财官、调候优先，却不知道落到一张真实排出来的盘上，这些原则怎么咬合、怎么取舍、哪里会打架。

命例库补的就是这一层：**每个命例取一组虚拟生辰，用脚本实跑出命盘，再严格按 SKILL 的 0-10 步走一遍完整推演**，每条结论后挂一个依据标签（出自哪个十神 / 宫位 / 用神 / 大运，可引 references）。它服务两类读者：

- **人**：看方法论怎么落地，对照自己的盘理解推演逻辑。
- **AI 模型**：作为「输入命盘、输出推演」的范例与对齐样本，学这套「先定旺衰用神、再断一切、孤证不立、句句带依据」的推演范式。

## 二、收录的 4 个命理原型

质量优先于数量。这 4 个原型覆盖了取用神最核心的几种分野，4 个日主（土 / 火 / 木 / 水）、4 种旺衰结构，不重复：

| 文件 | 原型 | 日主与盘性 | 演示要点 |
|---|---|---|---|
| [01_shenqiang_caiguan.md](01_shenqiang_caiguan.md) | **身强用财官** | 己土生午月，身强（同党 62%） | 日主得令、印比抱团而旺，财官有根透出，扶抑走「克泄耗」一路。富贵命最常见的结构。 |
| [02_shenruo_yinbi.md](02_shenruo_yinbi.md) | **身弱用印比** | 丁火生申月，身弱（同党 38%） | 日主失令、财官当道而身弱，扶抑走「生扶」一路，喜印生身比帮身。演示「印护身」在财旺局的关键，以及不能只数五行个数定强弱。 |
| [03_tiaohou.md](03_tiaohou.md) | **调候为急** | 乙木生子月，水寒木浮无火 | 冬生水寒、满盘印水而独缺火暖。演示**调候优先级压过扶抑**：量化身旺，第一用神却仍是火。呼应 references/03。 |
| [04_conge.md](04_conge.md) | **专旺润下格** | 癸水生子月，水一行独旺 | 日主旺极、全局无克破，顺其旺势立专旺格。演示特殊格的顺势取用，重点讲透**「见一点土便不成格，宁严勿凑」**的严谨，并辨析专旺与从格之别。呼应 references/05。 |

> 关于「从格」原型的取舍说明：04 最终落在**专旺（润下）**而非「从弱」。两者同属 references/05 的特殊格局、判法纪律一致（条件极严、一点破象即退回常格），专旺格的盘面更清纯、教学指向更明确，且文中已专门辟一节辨析专旺与从格的分界、并演示「见根即不成」的退格逻辑，覆盖了任务要求的核心教学点。

## 三、怎么读一个命例

每个命例固定按这个顺序展开，与 SKILL.md 的 0-10 步一一对应：

1. **主题与虚拟生辰**：这盘要演示什么原型，生辰是多少。
2. **命盘（脚本真实输出）**：`scripts/paipan.py` 的原始输出，放在代码块里，是后续一切推演的事实基础。**所有干支、五行力量数字都来自脚本，未经手改。**
3. **读盘定盘面**：日主、月令、关键刑冲合会。
4. **旺衰判定**：五看推理 + 身强 / 身弱 / 专旺 结论。
5. **用神喜忌**：调候 / 扶抑 / 顺势的取用逻辑 + 用·喜·忌·仇·闲。
6. **格局成败**：取格 + 成败救应 + 层次。
7. **大运流年要点**：各运吉凶 + 关键流年。
8. **分维度断语**：性格 / 事业 / 财运 / 婚姻 / 健康 / 学业 / 六亲，逐条带依据。
9. **调候趋避**：方位 / 颜色 / 行业 / 注意事项。
10. **综合总评**：3-5 句收束。

读法建议：**先看旺衰与用神两节，那是全盘的总开关**；后面的格局、大运、分维度断语，都是从用神延伸出来的。每条结论后的依据标签（如〔依据：午月得令 + 同党 62%〕），是这条断语的推理来源，可顺藤回查 references。

想自己复跑某个命例的命盘，照搬文中生辰即可：

```bash
python3 ../scripts/paipan.py <年> <月> <日> <时> <分> --gender <male|female>
```

若要复现与文中命盘块逐字一致的输出，再加一个参数：`--years 2024 12` 固定流年区间（流年表默认从当前年起，会随系统日期漂移）。神煞输出已按固定的传统吉凶次序排序，天然逐字可复现。每个命例的完整复跑命令已写在该文的「主题与虚拟生辰」一节。

## 四、阅读这套断语的口径

- **趋势化，非宿命**：全程用「易 / 倾向 / 利于 / 需注意」陈述倾向，不铁口断祸福。命理是基于模型的概率判断，真碰上婚姻、健康、重大决策，请把它当一个参考维度，结合现实理性拍板。
- **孤证不立**：每个重要结论尽量两处佐证（星 + 宫，或 原局 + 岁运），单一信号不下死断。
- **逻辑自洽**：旺衰结论与脚本给的五行力量、月令一致；用神与旺衰一致；断语与用神一致。这是命例库的生命线。

## 五、脱敏与免责声明

- **全部为虚构生辰**，不对应任何真实人物。命例主角一律用「某男 / 某女 / 案主 / 日主」代号，不涉真名。八字加生辰本身是强身份标识，命例库一律采用重新构造的虚拟生辰来演示，断语写**通用教学向**（这类盘面的人易倾向于……），不写可对号入座的具体生活事件。
- 每个命例文末附声明：**本例为虚构生辰的教学演示，非真人命造，仅供方法论研究。**
