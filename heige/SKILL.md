# 八字命理 · 精确排盘与系统推演

一套**可复现、有据可依**的四柱八字命理引擎。定位是「像做技术分析一样算命」：排盘交给脚本算准，推演按固定方法论逐层展开，**每个结论都标注依据**（出自哪个十神 / 宫位 / 用神 / 大运），孤证不立，中间推理全程透明。

## 核心原则（贯穿全程）

1. **排盘必用脚本**：干支推算交给 `scripts/paipan.py`，绝不手推。手推最易错在三处：年柱用错正月初一（应以立春）、月柱用错农历月（应以节气）、忽略真太阳时。
2. **方法论固定顺序**：排盘 → 定旺衰 → 取用神 → 判格局 → 析岁运 → 落十神六亲 → 分维度断语。**先定旺衰用神，再谈一切**，顺序不可乱。
3. **结论必带依据**：每条断语后注明推理链，如「事业偏技术创作（伤官透月干为用 + 食神生财）」。不写无依据的泛断。
4. **一象多看**：每个重要结论至少两处佐证（星 + 宫，或 原局 + 岁运），单一信号不下死断。
5. **展示推理过程**：本 skill 面向研究与深度分析，**完整呈现中间推演**（旺衰打分、取用逻辑、格局成败），不只给结论。
6. **命理是概率与倾向，非宿命**：用「易…、倾向、利于、需注意」等语气陈述趋势，落到可执行的趋避建议，而非铁口直断祸福。

## 第 0 步 · 收集与确认输入

必需信息：
- **出生日期**：阳历或农历，年、月、日。（默认按阳历；用户说农历/阴历则加 `--lunar`）
- **出生时间**：时、分。口语时间换算：早上8点=08:00，下午2点半=14:30，晚上11点=23:00。
- **性别**：男 / 女（决定大运顺逆排）。

可选信息：
- **出生地**（经度）：用于真太阳时校正，提升时柱精度。已知城市可用其经度（如北京116.4、上海121.5、广州113.3、成都104.1）。
- 不知道出生时间时：可只排年月日三柱，**但须声明**：时柱主子女、晚年、部分格局与时支神煞将缺失，结论存在局限，建议尽量回忆时辰。

确认信息无误后再排盘。信息有歧义（如农历阳历不明、闰月）必须先问清；用户生在农历闰月的，脚本里用负数月输入（如 `-2` = 闰二月）。

## 第 1 步 · 精确排盘（运行脚本）

在本 skill 所在目录下运行（脚本位于 skill 目录的 `scripts/` 下，装在别处按实际安装路径调整）：

```bash
python3 scripts/paipan.py <年> <月> <日> <时> <分> --gender <male|female> [选项]
```

常用选项：
- `--lunar`：输入按农历；**闰月用负数月表示**（如 `-2` = 闰二月）
- `--lng <经度>`：启用真太阳时校正（如 `--lng 113.3`），范围 -180~180，东经正、西经负
- `--tz <时区偏移>`：出生地时区（默认 +8，即东八区北京时间），配合 `--lng` 使用
- `--years <起始年> <年数>`：指定流年区间（如 `--years 2024 12`），默认从当前干支年（按立春分界）起 10 年
- `--json`：输出 JSON（需要结构化处理时）
- `--zi-sect <1|2>`：子时流派，1=晚子(23点)换日、2=不换日；出生在 23 点附近时声明所用流派

支持公历 1600-2200 年；起运岁数按**虚岁**口径（与大运列表一致）。

脚本输出包含：四柱干支 / 五行 / 藏干 / 天干十神 / 藏干十神 / 纳音 / 星运（长生十二宫）/ 旬空 / 节气与星座 / 胎元命宫身宫 / **地支刑冲合会与天干合冲（自动检出）** / **五行个数与缺失五行** / **五行力量加权分值与同党异党参考** / 神煞 / 大运（顺逆·起运·各步干支与十神）/ 流年。

**把脚本输出的命盘完整呈现给用户**，作为后续推演的事实基础。

## 第 2 步 · 读盘定盘面

从命盘提取关键要素，按 `references/01_paipan_jichu.md` 校验理解：
- **日主**（日干）：命主本人，记其五行、阴阳。
- **月令**（月支）：当令五行，旺衰第一权重。
- 天干透出的十神、地支藏干、通根情况。
- 地支之间的**刑冲合害会**（六合/三合/三会/六冲/相刑/六害），这是断动应、吉凶的关键，逐一标出。

## 第 3 步 · 判定日主旺衰（依据 `references/02`）

按「五看」综合判断，**展示推理**：
1. **月令得失**（权重最大）：日主在月支的长生状态，得令/失令。
2. **通根**：日主天干在四支有无同类根，坐下（日支）尤重。
3. **生扶**：印星生、比劫帮的力量。
4. **党众**：参考脚本给出的同党（印比）vs 异党（食伤财官）加权分值。
5. **定档**：身强 / 身弱 / 均衡 / 从格。

> 量化分值只是辅助，**月令权重远高于单纯计数**。临界（同党 45%–55%）时回到月令与通根细辨，不可只看百分比。先排除从格/专旺的极端情形（见 `references/05`），再按常格论。

## 第 4 步 · 取用神定喜忌（依据 `references/02` + `03`）

按五法择用，**优先级：调候 ≥ 扶抑 > 通关 > 病药；从格则顺势**：
1. **调候**：查 `references/03_tiaohou_qiongtong.md`，看日主生于该月所需调候用神是否在局中、是否有力。冬生无火、夏生无水须重点指出。
2. **扶抑**：身强取克泄耗（官杀/食伤/财），身弱取生扶（印/比劫）。
3. **通关 / 病药**：两行相战取通关；有忌神为病则取药。
4. 明确列出 **用神 / 喜神 / 忌神 / 仇神 / 闲神**，并说明取用理由。

用神是后续事业、财运、婚姻、健康、行业方位一切吉凶判断的总开关，务必论证清楚。

## 第 5 步 · 判定格局成败（依据 `references/05`）

1. **取格**：以月令本气透干定格，不透取中气余气透出者，再不透以月令本气论。
2. **成败**：顺用之格（官财印食）看是否得相神护卫、流通；逆用之格（杀伤刃枭）看是否得制化。指出成格、破格、或破后救应。
3. **特殊格局**：核对是否构成从格、专旺、化气、魁罡等，条件须严格（尤其从格，一点根气即不成）。
4. **定层次**：格成用旺 → 上；格成用弱或破而有救 → 中、起伏后成；格破无救 → 平常坎坷。

## 第 6 步 · 大运流年分析

- **大运**：脚本已排好顺逆、起运、各步干支与十神。逐步看大运五行**引动用神还是忌神**：引用神之运为顺境（事业财运婚姻向好），引忌神之运为逆境（对应领域受挫）。注意大运与原局的合冲（尤其冲提纲、冲用神）。
- **流年**：在当前/目标大运背景下，看流年干支与命局、大运的作用。重点关注：用神到位之年（机遇）、忌神当道之年（坎坷）、岁运并临、天克地冲、刑冲用神宫位之年。
- **应期**：把「会发生何事」（十神类象）落到「何时发生」（具体大运流年），给出可参考的时间段。

## 第 7 步 · 十神六亲落宫（依据 `references/04`）

- 标出各十神所在宫位（年月日时），对应人生阶段与领域。
- 取六亲星（男财女官为配偶、印为母、偏财为父、官杀/食伤为子女、比劫为兄弟），看其旺衰喜忌与所居宫位刑冲，断六亲缘分厚薄、助力损益。
- 日支为**夫妻宫**，重点看婚姻。

## 第 8 步 · 分维度断语（依据 `references/04`）

按需输出，每维度结论后**注明依据**：
- **性格**：日主五行阴阳底色 + 格局十神 + 最旺之神。
- **事业**：官杀（管理/公职）/ 食伤（技艺/创作/自由）/ 印（文教研究）为用的方向 + 适宜行业方位（取用神五行）。
- **财运**：财星旺衰、身能否任财、食伤生财、财逢冲夺；判富贵层次与得财方式、应财之运。
- **婚姻感情**：夫妻星 + 夫妻宫 + 星与日主合冲 + 桃花红艳，四象交叉。
- **健康**：五行过旺/受克绝对应脏腑（木肝胆·火心血·土脾胃·金肺·水肾），指出隐患与高发之运。
- **学业才华**：印星、食伤、文昌华盖、官印相生 / 伤官配印。
- **六亲**：父母、配偶、子女、兄弟缘分。

## 第 9 步 · 调候趋避与个性化调养（依据 `references/15` `16`）

把用神落到可执行，分趋避、色彩服饰、调养三层：

**趋避（环境与选择）**：
- **方位**：用神五行对应方位（木东·火南·土中·金西·水北），利发展的城市 / 方向。
- **颜色 / 数字 / 行业**：用神色（木青绿·火红紫·土黄棕·金白·水黑蓝）概览、宜从事的行业属性；色彩怎么穿戴见下「色彩服饰」层。
- **避忌**：忌神之运、之年的注意事项。

**色彩服饰（穿戴随身环境，依据 `references/16`）**：把用神色落到日常的衣着、配饰、首饰、随身物、居家办公环境，提升向、可执行。
- **认准标的**：穿戴用神喜神所属色、节制忌神过旺所属色。**缺的五行若为忌神则不穿不补，缺啥穿啥是误区。**
- **颜色为主轴**：颜色五行最有据；材质形状为辅；宝石玉石按呈现的主色取五行，只作参考，不神化、不承诺转运招财。
- **四个落地面**：穿（用神色做服装主色）、戴（首饰配饰取用神色与材质）、随身物（手机壳 / 包 / 车 / 文具，低门槛高频补色）、环境（居家办公主色调与灯光冷暖，呼应调候）。
- **守边界**：审美、得体、场合优先于五行，顺势调适、参考非转运。

**养生调养（作息饮食情志，依据 `references/15`）**：把命局的用神喜忌、缺失、过旺、寒暖燥湿、脏腑隐患，翻译成针对性的作息、饮食、情志、运动、起居建议，提升向、可执行。
- **认准标的**：只补用神喜神所属五行、节制忌神过旺所属五行。**缺的五行若为忌神则不补，缺啥补啥是误区。**
- **三步生成**：定标的（该扶 / 该泄的五行）→ 查五行调养总表译成作息饮食情志动养 → 叠加寒暖燥湿体质与脏腑隐患微调，每条带依据。
- **可结合习惯**：默认纯基于盘给建议；命主愿意提供作息饮食运动居住现状，则二次校准，给三到五条低门槛小调整。
- **守边界**：养生是辅助参考，务实非迷信、参考非医嘱，慢病急症孕产用药一律以医生为准。

## 第 10 步 · 综合总评

用 3–5 句收束：命局核心结构（日主+格局+用神一句话定性）、整体格局高低、一生大势起伏（哪些大运是关键转折）、最突出的优势与最需注意的短板。

## 第 11 步 · 可视化命书（可选，先征询）

文字推演（第 0–10 步）完整给出后，**主动问用户一句**：要不要把这份命书做成一页可视化 HTML 报告，方便保存、回看、分享。用户明确要，再做；不要就停在文字版。

动手前确认两件事：① 报告主题（默认「我的一生」）；② 信息口径，分享用途是否隐去真实姓名与生辰（脱敏）。

产出依据 `references/07_keshihua_baoshu.md`，记住四条铁律：
- **内容零改动**：HTML 只是文字推演的视觉化呈现，命盘、用神、断语必须与上文逐字一致，绝不为排版另造结论。
- **字体可靠性**：中文以思源宋体（Noto Serif SC）为骨架，字体栈必须以系统宋体兜底（`'Noto Serif SC','Songti SC','STSong',serif`），断网也不塌字；视觉层次靠字重 300→900 拉开，不依赖未必安装的书法字体。
- **配色取命局**：用神五行定主色调，全局一个朱砂、一个金做签名色，忌神色仅作点缀，不堆色。
- **同一口径**：趋势化、带依据、不恐吓，与文字版一致；页脚保留「仅供研究参考」。

> 可视化是锦上添花，不替代推演的严谨。没有完整的第 0–10 步，不直接出 HTML。

## 输出结构模板

```
【命盘】               ← 脚本完整输出
【日主与盘面】          ← 日主五行、月令、关键刑冲合会
【旺衰判定】            ← 五看推理 + 身强/弱/从 结论

【格局】               ← 取格 + 成败救应 + 层次
【大运流年】            ← 各运吉凶 + 关键流年 + 应期
【分维度详断】          ← 性格/事业/财运/婚姻/健康/学业/六亲，逐条带依据
【调候趋避与调养】      ← 方位·行业 + 色彩服饰（穿戴随身环境） + 作息饮食情志的养生建议
【综合总评】            ← 3–5 句收束
【可视化命书】（可选）    ← 征询用户后，依 references/07 出一页 HTML 报告
```

可按用户具体诉求裁剪：若用户只问「2027 年事业如何」，则聚焦排盘→用神→该运该年事业分析，不必全维度铺开，但旺衰用神这一根基步骤不可省。

## 断语风格

- 严谨、专业、有逻辑，像分析师拆解问题：先摆事实（盘面）、再推理（旺衰用神格局）、后结论（断语）。
- 趋势化表达（易/倾向/利于/需注意），不说死、不恐吓、不承诺。
- 命理结论是基于模型的倾向性判断；遇用户当真人生大事（婚姻、健康、重大决策）求决时，提示其作参考、结合现实理性决断。
