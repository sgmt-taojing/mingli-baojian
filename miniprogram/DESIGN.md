## 三、首页 — 功能大厅设计

### 3.1 页面结构（自上而下）

1. **Hero 横幅**：暗色背景 + 金色渐变标题「命理宝鉴」，副标题「知命改运 · 趋吉避凶 · 服务生活」，显示今日日期 + 干支
2. **每日箴言卡片**：展示当日命理箴言，点击可查看每日推送详情
3. **功能分区**：按 6 大板块分组，每个板块为一个卡片，卡片标题带板块图标和中文名，内部为 4 列网格布局
4. **功能入口项**：每项为圆形图标 + 功能名称（2-4字），点击跳转对应页面
5. **会员标识**：高级功能入口右上角显示「VIP」小角标（测试期不显示）
6. **底部声明**：传统文化传承声明 + 免责声明

## 四、排盘页 — 多术数排盘入口

### 4.1 页面结构

1. **标题区**：「命理排盘 · 推演天命」
2. **排盘类型选择**：横向滚动 Tab，6 种排盘类型
3. **表单区**：根据选择的排盘类型动态切换表单
4. **排盘结果区**：在当前页面下方展示结果
5. **历史记录区**：按排盘类型分类展示

### 4.2 排盘类型与表单

| 排盘类型 | 所需输入字段 | 结果展示内容 |
|---------|-------------|-------------|
| 八字排盘 | 姓名、性别、出生日期、出生时辰、出生地 | 四柱、藏干、十神、纳音、五行、神煞、大运流年 |
| 紫微斗数 | 姓名、性别、出生日期、出生时辰 | 十二宫位、星曜格局、四化飞星 |
| 奇门遁甲 | 姓名、性别、出生日期、出生时辰（或手动选时） | 天盘、地盘、八门、九星、八神、三奇 |
| 大六壬 | 姓名、性别、出生日期、出生时辰（或手动选时） | 四课、三传、天将、神煞 |
| 六爻占卜 | 姓名、性别（或铜钱摇卦交互） | 卦象、世应、六亲、六神、动爻 |
| 梅花易数 | 起卦方式选择（时间/数字），输入对应数字 | 本卦、互卦、变卦、体用生克 |

### 4.3 排盘结果展示

排盘结果在当前页面下方展示，采用卡片式布局：
- **基本信息摘要**：姓名、性别、出生信息一览
- **核心盘面**：根据排盘类型展示对应盘面图表
- **详细分析**：可折叠的分项解析
- **大运流年**：表格形式展示（八字）/ 宫位流转（紫微）
- **操作按钮**：保存记录、分享、查看详细解读

## 八、二级页面详细设计

### 8.1 八字类二级页面

#### 8.1.1 起名改名页 `pages/naming/naming`

- **功能子 Tab**：[起名] [改名] [测名]
- **起名**：输入姓氏、性别、出生信息、五行偏好、字数、风格 → 生成推荐名字列表（含五格评分、五行配置）→ 点击查看详细分析（三级页面）
- **改名**：原名分析 + 新名推荐
- **测名**：输入姓名 → 五格数理分析结果

#### 8.1.2 合婚分析页 `pages/hehun/hehun`

- **表单**：男方信息（姓名/出生日期/时辰）+ 女方信息（同上）
- **结果**：匹配度百分比 + 五行互补/属相合配/日柱关系 + 详细分析（性格匹配、婚姻运势、注意事项、化解建议）

#### 8.1.3 体质辨识页 `pages/constitution/constitution`

- **表单**：出生日期 + 时辰（自动推算五行体质）
- **结果**：五行体质类型 + 体质特征 + 养生方案（饮食建议、运动建议、起居调养、情志调节）

### 8.2 占卜类二级页面

#### 8.2.2 奇门遁甲页 `pages/qimen/qimen`
