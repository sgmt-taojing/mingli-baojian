📖 使用说明

① <ml-toast> — 轻量消息提示

属性类型默认说明
typesuccess|error|warn|infoinfo提示类型（背景色随之变化）
durationnumber (ms)2400自动关闭延时
positiontop|bottom|centertop屏幕位置

方法：show(msg, type) · success(msg) · error(msg) · warn(msg) · info(msg) · hide()
CSS 变量：--ml-bg · --ml-ink · --ml-success · --ml-error · --ml-warn · --ml-info

旧 API 兼容：引入 toast.js 后自动创建全局 window.toast 对象， 支持 toast.show(msg, type) / toast.success() 等链式调用； 同时兼容 shToast(msg) 旧函数名。旧页面无需改动即可生效。

② <ml-modal> — 模态弹窗

属性类型默认说明
titlestring—弹窗标题（不设则隐藏标题栏）
opentrue|falsefalse是否打开
sizesm|md|lgmd弹窗宽度（420/640/840px）
close-on-backdroptrue|falsetrue点击遮罩是否关闭

方法：open() · close(source?)
事件：open（detail: {}） · close（detail: {source:'x'|'backdrop'|'escape'|'api'}）
CSS 变量：--ml-bg · --ml-panel · --ml-ink · --ml-border

旧 API 兼容：自动检测 .modal / .modal-overlay / [id$="Modal"] 元素并升级为 <ml-modal>，旧代码 display='flex'/'none' 仍可工作。 暴露 window.MlModal 和 window.__mlModalUpgrade() 辅助。

③ <ml-tab> + <ml-tab-pane> — 标签页切换

属性宿主元素类型 / 默认说明
activeml-tabnumber / 0当前激活面板索引
positionml-tabtop|bottom / top标签栏位置
alignml-tableft|center|right / left标签栏对齐
labelml-tab-panestring按钮文字（必填）
iconml-tab-paneemoji/string图标前缀，与 label 拼接
disabledml-tab-panetrue禁用该标签

方法：setActive(index) · activeIndex（getter）
事件：tab-change（detail: {index, label, prevIndex}）
CSS 变量：--ml-primary · --ml-muted · --ml-bg · --ml-border · --ml-ink-on-primary

旧 API 兼容：若页面不存在 window.switchTab，自动注册一个兜底版本， 兼容旧式 .tab-btn + .tab-panel 模式。 暴露 window.MlTab 和 window.MlTabPane。
💡 通用规则：所有组件采用 Custom Elements v1 规范，Shadow DOM 封装样式， 支持全局 CSS 变量覆盖主题。引入对应 JS 文件即自动注册，无需额外初始化。 旧代码通过 polyfill 层自动兼容，无需改动业务逻辑。
