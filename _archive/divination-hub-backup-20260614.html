📖 风水基础理论

气论 · 阴阳 · 五行 · 八卦 - 风水学的四大理论支柱

一、气论:风水的核心

气,是风水学的核心概念，指宇宙万物运行的根本能量。《葬书》(郭璞)开宗明义:"葬者，乘生气也"。生气即天地之间流动的正能量，是万物生长、人类繁荣的根本。风水之术，本质是"乘生气、避死气、化煞气"。

📜 经典出处 - 《葬书》(郭璞):
"葬者，乘生气也。气乘风则散，界水则止。古人聚之使不散，行之使有止，故谓之风水。"
"气感而应，鬼福及人。是以铜山西崩，灵钟东应。"
🌿 生气

生机勃勃、向上向善之气。表现为:阳光充足、空气流通、水质清澈、草木茂盛。住宅得生气，则人丁兴旺、财运亨通、身体健康。

💀 死气

衰败、停滞、腐朽之气。表现为:阴暗潮湿、空气污浊、死水积聚、草木枯败。住宅犯死气，则家运衰退、疾病缠身、财运不济。

⚡ 煞气

急骤、尖锐、冲突之气。表现为:尖角冲射、直路冲宅、反弓水、高压电塔。煞气分形煞(可见之形)与气煞(不可见之气),皆需化解。

二、阴阳平衡:风水的根本法则

阴阳是宇宙万物的两大对立统一力量。《青囊经》(黄石公)曰:"阳生于阴，阴生于阳。阴阳相济，万物化生。"风水布局的核心，是追求阴阳的动态平衡，而非绝对的阴阳对等。

📜 经典出处 - 《青囊经》(黄石公):
"天尊地卑，阳奇阴偶。一六共宗，二七同道，三八为朋，四九为友，五十同途。"
"阴阳相乘，有所自来。阴阳相度，有所往还。"

🏠 住宅阴阳应用:

  • ✅ 前阳后阴:前方(明堂)宜开阔明亮(阳),后方(靠山)宜稳固安静(阴)
  • ✅ 左阳右阴:左侧宜略高或活跃(阳),右侧宜略低或沉稳(阴)-- "左青龙、右白虎"
  • ✅ 昼阳夜阴:白天采光充足(阳),夜晚安静暗淡(阴)
  • ✅ 动静分区:客厅/厨房宜动(阳),卧室/书房宜静(阴)
  • ⚠️ 阴阳失调:全阳(燥热、不安)→ 增阴(加水景、用冷色);全阴(阴冷、抑郁)→ 增阳(加灯光、用暖色)

三、五行与方位:风水的空间密码

五行(木火土金水)是风水的空间编码系统，将方位、色彩、材质、人体器官一一对应。《阳宅十书》(王君荣)详述了五行与住宅布局的关系:"木主东方，其色青；火主南方，其色赤；土主中央，其色黄；金主西方，其色白；水主北方，其色黑。"

📜 经典出处 - 《阳宅十书》(王君荣):
"东方属木，木生火，故东房宜人居住。南方属火，火生土，故南房宜作厨房。中央属土，土生金，故中堂宜宽敞。西方属金，金生水，故西房宜作书房。北方属水，水生木，故北房宜作寝室。"

四、八卦与方位:风水的方位坐标系

八卦是风水的方位坐标系，分为先天八卦(伏羲八卦)与后天八卦(文王八卦)。风水实践中用的是后天八卦,即:坎北、离南、震东、兑西、乾西北、坤西南、艮东北、巽东南。

☯ 先天八卦(伏羲八卦)

先天八卦讲"对待",即天地定位、山泽通气、雷风相薄、水火不相射。其方位为:乾南、坤北、离东、坎西、震东北、兑东南、巽西南、艮西北。

应用:先天八卦主要用于理气(抽象的气场计算),如玄空飞星的"元旦盘"即基于先天八卦数(乾9、坤1、巽2、兑7、艮6、离3、坎7、震8--简化记为"乾离坎震各自主宰")。

☯ 后天八卦(文王八卦)

后天八卦讲"流行",即万物生长收藏的运行次序。其方位为:坎北、离南、震东、兑西、乾西北、坤西南、艮东北、巽东南。

应用:后天八卦是风水实践的核心坐标系,八宅明镜、玄空飞星、家居布局皆以后天八卦为基准。

一、八宅明镜概述

八宅明镜,相传为唐代堪舆大师杨筠松(杨公)所著，是八宅派风水的经典典籍。其核心理论是:将人的命卦分为"东四命"与"西四命"两大类，将住宅分为"东四宅"与"西四宅"两大类，命卦与宅型匹配则吉，不匹配则凶。

📜 经典出处 - 《八宅明镜》:
"夫宅者，乃阴阳之枢纽，人伦之轨模。凡宅，东四命人居之则吉，西四命人居之则凶；西四命人居之则吉，东四命人居之则凶。"
"命卦以出生年份推算，男用九星逆数，女用九星顺数。"

二、东四命与西四命计算方法

命卦的计算方法是八宅风水的核心技能。注意:2000年及以后出生者，计算方法有所不同(用9减代替用10减)。

👨 男命卦计算法

步骤:

1. 出生年份四位数相加，得和A

2. 若A≥10,再将A的各位数字相加，得和B

3. 1900-1999年生:用10减B,得命卦数

4. 2000年及以后生:用9减B,得命卦数

5. 命卦数对应八卦(1坎、2坤、3震、4巽、5男命寄坤、6乾、7兑、8艮、9离)

例:1990年生男 → 1+9+9+0=19 → 1+9=10 → 1+0=1 → 10-1=9 → 9离 → 东四命

👩 女命卦计算法

步骤:

1. 出生年份四位数相加，得和A

2. 若A≥10,再将A的各位数字相加，得和B

3. 1900-1999年生:B即为命卦数(若B=10,命卦数为1)

4. 2000年及以后生:用B+5,得数若≥10再各位相加

5. 命卦数对应八卦(1坎、2坤、3震、4巽、5女命寄艮、6乾、7兑、8艮、9离)

例:1990年生女 → 1+9+9+0=19 → 1+9=10 → 1+0=1 → 1坎 → 东四命

厨房属火,是财库("灶位旺财")。厨房布局得当，则家人健康、财运稳定。

位置:厨房宜位于住宅后方("财库在内"),不宜在正中或前方。

四、卫生间风水:化泄为安

卫生间属水,且为排污之处,气场较污浊。卫生间布局得当，则污气不侵；布局不当，则病气蔓延。

📍 位置禁忌

忌:卫生间位于住宅中央("污气居中，全家受染")。

忌:卫生间正对大门/卧室门/厨房门("门冲")。

宜:卫生间位于凶方(以毒攻毒，镇压凶气)。

1. 保持干燥通风:开窗/排气扇常开，减少湿气污气。

2. 化煞:门外有煞气(如尖角/路冲),挂八卦镜化解(注意:八卦镜宜对外，不宜对内)。

⚠️ 葫芦禁忌

一、三才五格算法详解

姓名学中的「五格」指的是天格、人格、地格、外格、总格，源自日本熊崎健翁的《姓名学》,后传入中国并与传统五行理论结合。《梅花易数》亦有以笔画数理推断吉凶之法。

1. 天格计算:天格代表祖先运，对人生影响较小。单姓计算公式为「姓的康熙笔画 + 1」。例如:王(4画) → 天格 = 4 + 1 = 5。

2. 地格计算:地格代表前运(36岁前),影响基础运。单名计算公式为「名的康熙笔画 + 1」;双名计算公式为「名的第一字笔画 + 名的第二字笔画」。例如:李明(7+8=15) → 地格 = 15。

3. 人格计算:人格代表主运，是姓名分析的核心。计算公式为「姓的末字笔画 + 名的第一字笔画」。例如:马云(马17画+云12画) → 人格 = 17 + 12 = 29。

4. 外格计算:外格代表副运，影响社交与外在表现。计算公式为「总格 - 人格 + 1」。例如:总格29 - 人格29 + 1 = 1。

5. 总格计算:总格代表后运(36岁后),影响一生总运势。计算公式为「姓名全部字的康熙笔画之和」。例如:马云(17+12=29) → 总格 = 29。

三、五行补益分析

根据生辰八字推算五行喜用神，是姓名五行补益的核心。《三命通会》云:「八字贵在中和，偏旺偏衰皆不为美。」姓名五行补益原则是「用神字 > 忌神字」,即优先选择与喜用神五行相同的字。

五行对应笔画尾数:

木:笔画尾数 1、2(如 1画、2画、11画、12画...)
火:笔画尾数 3、4(如 3画、4画、13画、14画...)
土:笔画尾数 5、6(如 5画、6画、15画、16画...)
金:笔画尾数 7、8(如 7画、8画、17画、18画...)
水:笔画尾数 9、0(如 9画、10画、19画、20画...)

五行补益原则:

1. 用神补益:选择五行与喜用神相同的字，如喜木可选「林」(8画，木)、「梓」(11画，木)
2. 忌神回避:避免选择五行与忌神相同的字，如忌金应避免「锋」(15画，金)、「鑫」(24画，金)
3. 五行流通:姓名三才五格的五行应相生而非相克，如「木生火、火生土」为吉
4. 平衡调和:若八字某五行过旺，可用克泄之五行调和，如木过旺可用金克或火泄

实例分析:某人八字喜用神为「木、火」,则取名时宜选木、火五行的字，如「林」(木)、「炎」(火);避免金、水五行的字，如「锋」(金)、「海」(水)。

四、音律分析

姓名的音律直接影响他人呼叫时的感受，好的姓名应读起来朗朗上口、避免拗口和谐音。

1. 声母搭配:声母是拼音开头的辅音(如 zh、ch、sh、b、p、m、f 等)。好的姓名应避免声母完全相同导致拗口。例如:「张志忠」(zh-zh-zh) 声母全同，读起来费劲；「李玲丽」(l-l-l) 同理。建议声母错落有致，如「张明辉」(zh-m-h) 声母各异，读起来顺口。

2. 韵母搭配:韵母是拼音中声母后的部分(如 a、o、e、i、u、ü 等)。避免韵母完全相同导致单调。例如:「王芳芳」(ang-ang-ang) 韵母全同，缺乏韵律感；「李思琪」(i-i-i) 同理。建议韵母有变化，如「张雨涵」(ang-yu-han) 韵母各异，读起来有节奏感。

3. 声调搭配:普通话有四个声调(阴平、阳平、上声、去声)。好的姓名应平仄搭配，读起来抑扬顿挫。一般建议姓氏与名字的声调错落有致，避免连续三个字同声调。例如:「张平平」(阴平-阳平-阳平) 尚可；「李磊磊」(上声-上声-上声) 读起来很拗口。

4. 谐音警示:避免姓名产生不雅的谐音。例如:「杜子腾」谐音「肚子疼」、「沈京兵」谐音「神经病」、「朱逸群」谐音「猪一群」、「秦寿生」谐音「禽兽生」、「刘产」谐音「流产」、「范统」谐音「饭桶」、「魏生津」谐音「卫生巾」、「费彦」谐音「肺炎」、「胡丽晶」谐音「狐狸精」。取名后务必多读几遍，检查是否有不良谐音。

优(90-100分):声母、韵母、声调均有变化，读起来朗朗上口
良(70-89分):声母或韵母有变化，但有轻微瑕疵
中(50-69分):声母、韵母或声调有明显问题，读起来稍拗口
差(0-49分):声母、韵母、声调均有问题，读起来很拗口

五、寓意分析

姓名的寓意是父母对孩子的期望和祝福，好的姓名应有美好的寓意、典雅的出典、适合的性别指向和时代感。

1. 字义解释:每个名字用字都应有正面的解释。例如:「伟」意为高大、杰出；「芳」意为芳香、美好；「娜」意为柔美、优雅；「敏」意为敏捷、聪慧；「静」意为安静、沉稳；「杰」意为杰出、优秀；「慧」意为聪慧、智慧；「浩」意为浩大、广阔；「博」意为博大、丰富；「睿」意为睿智、聪明；「涵」意为涵养、包容；「瑾」意为美玉、品德。

2. 出处典故:引经据典的姓名更有文化底蕴。例如:「浩然」出自《孟子·公孙丑上》:「吾善养吾浩然之气」;「明德」出自《大学》:「大学之道，在明明德」;「思齐」出自《论语》:「见贤思齐焉」;「婉清」出自《诗经·郑风》:「清扬婉兮」;「子衿」出自《诗经·郑风》:「青青子衿，悠悠我心」;「嘉卉」出自《诗经·小雅》:「山有嘉卉，侯栗侯梅」。

3. 性别适配:男名宜阳刚、大气，如「浩宇」、「鹏飞」、「峻熙」;女名宜柔美、雅致，如「婉清」、「诗涵」、「雅琪」。避免男用女名或女用男名导致性别混淆。

4. 时代适配:避免过时字(如「寿」、「福」、「财」等过于直白的字);避免生僻字(如「龘」、「靐」、「齉」等，会带来生活不便);避免过于流行的字(如近年来「梓」、「涵」、「轩」使用过多，缺乏个性)。建议选择经典且有新意的字，如「砚」、「溯」、「曜」、「浔」等。

良(70-89分):字义正面、有出典或寓意、性别基本适配
中(50-69分):字义一般、缺乏出典、性别适配度一般

一、五格计算方法

1. 天格（Tiān Gé）

单姓:姓氏笔画 + 1（外格之理，天圆地方，姓为天，加1为全）

复姓:姓氏两字笔画之和

例:「王」4画 → 天格 = 4+1 = 5（土）

2. 人格（Rén Gé）

单姓单名:姓氏笔画 + 名字笔画

单姓双名:姓氏笔画 + 名字第一字笔画

复姓:姓氏第二字 + 名字第一字

例:「王明」→ 人格 = 4+8 = 12（木）

3. 地格（Dì Gé）

单名:名字笔画 + 1

双名:名字两字笔画之和

复名（3字以上）:名字各字笔画之和

例:「明」8画 → 地格 = 8+1 = 9（水）；「志远」→ 地格 = 7+17 = 24（火）

4. 外格（Wài Gé）

公式:外格 = 总格 - 人格 + 1

含义:外在形象、社交能力、与外部环境的关系

5. 总格（Zǒng Gé）

公式:姓名所有字笔画之和

含义:一生总运、晚年运势、整体成就

二、公司名特殊性

两字名（如:阿里）

天格 = 第一字+1；人格 = 第一字+第二字；地格 = 第二字+1；总格 = 两字之和；外格 = 总格-人格+1

三字名（如:腾讯云）

天格 = 第一字+1；人格 = 第一字+第二字；地格 = 第二字+第三字；总格 = 三字之和；外格 = 总格-人格+1

四字名（如:阿里巴巴集团）

天格 = 第一字+第二字；人格 = 第二字+第三字；地格 = 第三字+第四字；总格 = 四字之和；外格 = 总格-人格+1
