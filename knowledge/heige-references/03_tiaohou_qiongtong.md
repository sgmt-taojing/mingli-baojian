# 调候用神速查（穷通宝鉴 / 栏江网精要）

> 按**日主 × 出生月令（节气月，非农历月）**查调候所需天干。调候与扶抑冲突时，气候偏枯者优先调候。
> 表中用神按重要性排序，前者为主。这是「气候层面」的补救，需与 `02` 的扶抑、格局综合定夺。

## 十干调候口诀（总纲）

- **甲木**：春取丙暖癸滋，夏专癸水润，秋用庚劈丁炼，冬需丁丙庚解寒。
- **乙木**：花草之木，喜丙暖癸润；冬月寒谷回春，专用丙火。
- **丙火**：阳精，以壬水显其辉，庚金为壬之源；冬月身弱需甲戊。
- **丁火**：灯烛之火，喜甲木为引、庚金劈甲；夏月火旺用壬癸济之。
- **戊土**：高燥之土，喜丙暖、甲疏、癸润，三神配齐为贵。
- **己土**：田园湿土，喜丙火暖燥、癸水润苗；冬月专取丙火。
- **庚金**：顽金需丁火锻炼、甲木引丁；夏火旺用壬癸制火存金。
- **辛金**：珠玉之金，专喜壬水淘洗增辉；冬月寒，需丙火暖局。
- **壬水**：江河之水，旺则需戊土堤防、丙火调候；弱则用庚辛生。
- **癸水**：雨露之水，喜庚辛为源；冬月冰冻，必用丙火解冻。

## 用法要点

1. **先看调候是否到位**：表中用神在原局透出且有力 → 富贵之兆；全无 → 格局再好也减层次，待大运补则发。
2. **寒暖燥湿四字诀**：金水多偏寒湿（喜火土）、木火多偏燥热（喜水金）；土过燥（戌未月、火多）喜水润，土过湿（辰丑月、水多）喜火暖。
3. **调候用神也要落到地支有根**：仅天干虚露、地支无根，调候力薄。
4. 与扶抑结论交叉验证：若调候用神恰与扶抑用神同向（如冬生身弱用丙火，既暖局又生扶/通关），则用神明确、命局清纯。
