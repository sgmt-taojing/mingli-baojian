# 《命理约言》理性论法：少玄学，重五行本质

> **校准用途。** 本篇提炼清·陈素庵《命理约言》的理性派论法，给推演一个祛魅视角：以日主旺衰、生克扶抑、用神为本，神煞纳音胎元小运等旁支慎用或不用。遇命局被神煞名目带偏、被怪异格局牵扯时，回到本篇做减法校准。本篇与 `02_wangshuai_yongshen.md` 互补：02 给取用神的完整操作框架，本篇给取用神的精炼总纲与理性边界，不重复五法细节。

## 〇、版本与作者（出处底账）

- **作者**：陈素庵（1605–1658），名之遴，字彦升，晚号素庵老人，浙江海宁人。明崇祯十年（1637）进士，入清官至礼部尚书、弘文院大学士，后获罪流徙而卒。
- **整理流通**：民国嘉兴韦千里选辑评注，定名《精选命理约言》，约1933年成稿、1935年印行，遂广为流传。本篇引文以韦氏流通本为据。
- **全书结构**：分四卷，卷一为「法」、卷二为「赋」、卷三为「论」、卷四为「杂论」，后附张神峰《辟五行诸谬论》。**「法」立操作规程，「论」辨干支神煞之理，「杂论」与附论行祛魅之实。**（出处：《命理约言》卷目；韦千里序）
- **存疑标注**：不同流通本对各卷篇数记述略有出入（如卷三「论」、卷四「杂论」的具体则数各本不一），凡本篇未能逐字核对原文者均标「存疑」，短引名句照录，长段古文不照抄。

## 一、理性论法总纲

陈素庵的方法论可压缩成一句**核心断语**与一套**看命次第**。

### 总纲一句话

> **「看命大法，不过生克扶抑而已。」**（卷一·看命总法）

一切富贵贫贱寿夭婚病，皆由五行生克与扶抑强弱推出，不假外求于神煞吉凶之名。

### 看命次第（先旺衰用神，后格局岁运）

照《看命总法》的操作顺序，可整理为五步速查：

| 步骤 | 动作 | 要点（出处：卷一·看命总法） |
|---|---|---|
| 1 排柱定日干 | 「列下四柱，先看日干是何五行」 | 先锚定日主，万事以日主为我 |
| 2 看月支定旺衰 | 「随看月支，或是生我克我，或是我生我克」 | 月令司权，定日主得令失令，旺衰之根在此 |
| 3 月气透干取格 | 月支本气透干则取为格（寅透甲、午透丁之类）；本气遭克不透，则取所藏次神；再不堪，则取别干支势盛者为格 | 格局由五行月令自然生成，不靠神煞名目硬安 |
| 4 扶抑定用神 | 「强者抑之，弱者扶之」 | 用神即调中和之神，详见 `02` 扶抑法 |
| 5 局不堪则取运 | 「不能扶抑者，以运扶抑之」 | 原局补不足者待岁运补，引出大运流年的吉凶 |

**次第不可倒置。** 先把日主旺衰、用神喜忌定死，再论格局成败、岁运吉凶。神煞放在最后，且只作细节点缀，与 `06_shensha.md` 的使用原则一致。

### 中和为贵的总判

> **「日主最贵中和。」「清而有神者贵，浊而无气者贱；纯粹中和者贵而安，奇怪偏驳者贵而危。」**（卷一·看命总法）

- 趋势上，命局越趋中和，越易吉多凶少；偏枯太过，纵有贵气也易带危。
- 推演时遇「奇格异局」的诱惑，先用中和这把尺子量一量，偏驳者断语宜留余地，不铁口言贵。

## 二、取用神纲领（与 02 互补的精炼之处）

`02` 已给扶抑、调候、通关、病药、从格五法的完整操作。本节只补《命理约言》对「用神」二字的纲领性提炼，给取用一个更本质的定位。

### 用神是命之精神

> **「命以用神为紧要。」「命譬之身，用神譬之身之精神；精神厚则身旺，精神薄则身衰。」**（卷一·看用神法）

- 用神为当下最能令此局趋中和的那一点五行力量，随局而定，无固定十神。
- 衡量层次：用神得力、有生扶、不受冲克伤损者，命局精神厚，趋向富贵安稳；用神虚浮受制者，精神薄，层次易低、易危。

### 取用的尺度（过与不及都要回调）

陈素庵论扶抑，强调一个**动态分寸**，可作 02 扶抑法的细化补充：

| 情形 | 处理（出处：卷一·看用神法，文意提炼） |
|---|---|
| 日主弱 | 扶之者为用神 |
| 扶之太过 | 反成抑，需回调 |
| 扶之不足 | 仍当更扶 |
| 日主强 | 抑之者为用神 |
| 抑之太过 | 反伤本根，需生扶回护 |
| 抑之不足 | 仍当更抑 |

**要点**：取用要看「扶到几分、抑到几分」才中和，是一个动态过程。过则反向微调，这是判断岁运吉凶时尤其要带上的尺子。

### 取用的态度

> **「取用神之法，虽当专一而不玄眩，宜变通而不拘。」**（卷一·看用神法）

- **专一**：一局之中主用神宜清宜定，不东拉西扯凑吉。
- **变通**：原局取不到，许从岁运补；此格不就，许转看他神。
- **不玄眩、不拘**：祛魅总纲在取用一步就立住了，少在玄虚名目上绕。

## 三、看格局法「约论」要点

> **「格局有正有变。正者，五行之常理也。」**（卷一·看格局法）

陈素庵把繁杂格局收束为「正格 + 变格」两类，砍掉大量旁门杂格，这是其「约」字的体现。

### 正格（六格，由月令五行自然生成）

正官、偏官（七杀）、印、财、食神、伤官六格。**正格少有不相兼者**，常见配搭：

| 正格 | 常见相兼用法（出处：卷一·看格局法） |
|---|---|
| 正官格 | 兼印或兼财 |
| 偏官（七杀）格 | 兼印或兼财（制化为要） |
| 印格 | 兼官或兼煞 |
| 财格 | 兼官或兼煞 |
| 食神格 | 用煞或用财 |
| 伤官格 | 取印或取财 |

### 变格（五类，精审方立）

> **「曰从，曰化，曰一行得气，曰两神成象，曰暗冲，曰暗合。」**（卷一·看格局法；其中从、化、专旺等已见 `02` 从格/专旺部分）

变格成立条件严，陈素庵态度是「变格更宜精审」。**趋势上**，疑似变格而原局尚有一分根气或克破者，宜退回正格按扶抑论，不轻许从化。

### 格局无定吉凶（最具理性色彩的一句）

> **「得时得局，煞伤可以富贵；失时失局，官印可以贫贱。」**（卷一·看格局法）

- 七杀伤官未必凶，正官正印未必吉。**吉凶系于得时得局与否（即旺衰用神），不系于十神名目的好坏。**
- 这一条直接破「见官就贵、见杀就凶」的名目迷信，是断格局时的祛魅锚点。

## 四、看六亲法「约论」要点

> 旧法以正印为母、偏财为父、正财为妻、官杀为子女等，按十神硬派六亲。**陈素庵指出这套硬派存在逻辑矛盾**（如偏财既派为父又派为偏妻），主张回到生克扶抑综合判断。（出处：卷一·看六亲法，文意提炼；对旧法之辨亦见后世《子平真诠》申论，**存疑：原文逐字待核**）

### 推六亲的理性原则

1. **十神定位只是参照，不可机械套用。** 父母妻财子女的吉凶厚薄，要看其对应之神在局中是喜是忌、得生扶还是受冲克，而非只认名目。
2. **宫位与十神参看**：年月日时各宫所主六亲，结合该柱干支的旺衰生克定亲缘厚薄。
3. **断语趋势化**：六亲缘分的厚薄早晚，宜言「倾向」「易」「需注意」，不铁口断生死刑克。

## 五、看性情 · 贫富 · 贵贱 · 寿夭诸法「约论」

陈素庵立《看性情法》《看贫富法》《看贵贱法》《看寿夭法》等专篇，皆从五行十神与中和推出，不假神煞。提炼可执行要点如下（出处：卷一各专法，文意提炼；**部分篇名各本略有出入，存疑**）：

### 看性情

- 性情由日主五行之性与当令十神之气合成：木主仁而条达，火主礼而急明，土主信而厚重，金主义而刚断，水主智而灵动。
- 中和则性情和顺，偏枯太过则性情偏激。印重者文静、伤官旺者傲、官杀重而无制者多畏忌。**断性情宜言倾向，不作定论。**

### 看贫富

- **财为养命之源，看财之有无、真假、得地与否，以及日主能否任财。**
- 趋势：身旺财旺有库有流通者，倾向富；财多身弱（财不胜任）或财轻劫重者，倾向不聚财。

### 看贵贱

- **贵由清纯中和、官杀印有情、用神得力而来，不由神煞名目而来。**
- 趋势：用神清、格局正、岁运扶用者，利于贵显；浊乱无气、用神受伤者，倾向平常。

### 看寿夭

- 看日主元气厚薄、五行有无大冲大克大战、用神是否被连根拔。
- 趋势：气厚流通、无毁折之象者利于寿；偏枯无救、用神被克尽、岁运再引大冲者需注意寿元之险。**此处尤忌铁口，只标风险趋势，不预言生死时点。**

## 六、《辟谬》祛魅：神煞纳音小运胎元的理性批判

这是《命理约言》理性派价值最集中处。陈素庵在卷三「论」诸篇与卷四「杂论」、并附张神峰《辟五行诸谬论》，对子平正法之外的旁支系统逐一祛魅。**总精神**：以五行生克旺衰用神为本，神煞等慎用，无理据者不取。

### 祛魅总原则（务必内化为断命纪律）

1. **十神是关系之名，不是吉凶之神。** 官未必得官，财未必得财；得时得位，煞伤亦富贵，失时失位，官印亦贫贱。（卷三论意，参 §3 末句）
2. **神煞为辅，不主断。** 与 `06_shensha.md` 第四节使用原则完全一致：先断格局用神，神煞只作细节点缀。
3. **无理据、无生克根基者，从简或不取。** 凡只靠口诀歌赋、不讲五行生克的体系，皆列为可疑。

### 各旁支系统的批判清单

| 旁支系统 | 陈素庵/张神峰的理性判断 | 出处与核验状态 |
|---|---|---|
| **天乙贵人等吉神煞** | 单凭贵人不可定贵格；「贵人头上载财官」「日贵格」之说各只一端，不足独立成格，须全观四柱旺衰用神 | 卷三·贵人论，**已核引意**：「此一端未可决为贵格」 |
| **天月二德** | 非绝对有效；「若二德自坐行冲，则亦无力」，逢冲破即减力 | 卷三·天月二德论，**已核引意** |
| **贵人起例本身** | 诸神煞起例来历多不明白，「皆不足深究」，不必在名目来源上纠缠 | 卷三·贵人论，**已核引意** |
| **纳音五行** | 婁景以炉中、海中、大林、路旁等编纳音歌诀便于成诵，后世误以为实然；论命当用正五行生克，纳音无生克正理之据 | 卷四·杂论，**已核引意**（短引「配纳音五行为歌……后世谓为实然」） |

| **合婚煞法** | 旧合婚书「止以年一字，犯某日某时为言」，仅凭年支单字吓唬人，缺生克正理 | 卷四·杂论，**已核引意** |
| **怪异选格诸书** | 《三车一览》《望斗真经》《兰台妙选》等「俱不论生克正理，漫以江山水石风雨立说」，弃正理而取怪论，不可从 | 卷四·杂论，**已核引短句** |

## 八、一页速记

1. **看命大法不过生克扶抑而已**：先日主旺衰用神，后格局岁运，次第不可倒。
2. **用神是命之精神**：厚则旺、薄则衰；扶抑过则回调，专一而不玄眩。
3. **格局收束为正格+变格**：得时得局，煞伤可富贵；失时失局，官印可贫贱。
4. **六亲十神定位只作参照**，结合喜忌生克断厚薄，趋势化不铁口。
5. **辟谬祛魅**：神煞为辅不主断；纳音、生肖、合婚煞法、怪异选格诸书无生克正理者存疑或不取；小运胎元命宫属旁支（存疑待核）。
6. **引用纪律**：已核引意者可用，标「存疑」者只作理性派倾向表述，绝不杜撰古文长段。
