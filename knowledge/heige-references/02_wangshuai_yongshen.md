# 旺衰判断 与 取用神五法：命理推演的核心

> 全局成败系于此。**先定日主旺衰，再定用神喜忌**，后续事业财婚健康一切断语皆由用神延伸。

## 一、判定日主旺衰（五看）

按权重从大到小：

### 1. 看月令（得令 / 失令）：权重最大
日主生于何月，是否当令。月令司权，占旺衰判断的三到四成。
- **得令**：日主五行在月支处于旺、相状态。如木日生于寅卯月（木旺）、生于亥子月（水生木，相）。
- **失令**：日主处于休、囚、死。如木日生于申酉月（金克木，囚死）。

五行随季节旺相休囚死：**当令者旺，我生者相，生我者休，克我者囚，我克者死。**

### 2. 看通根（得地 / 失地）
日主天干在四地支中有无同类藏干为「根」。
- 通根本气（如甲见寅、庚见申）为强根；通根中气、余气较弱；通到墓库需冲刑引动。
- **坐下（日支）通根**力量最重，是日主的「自坐」根基。

### 3. 看生扶（得生 / 得助）
- 印星生我（如金日见土）、比劫帮我（如金日见金），数量与力量。
- 透干通根的印比力量大，仅藏不透者力小。

### 4. 看党众（同党 vs 异党）
- **同党（扶日主）**：印星 + 比劫。
- **异党（耗日主）**：食伤 + 财星 + 官杀。
- `paipan.py` 已给出五行加权分值与同党/异党总分作量化参考。**但量化只是辅助，月令权重远高于单纯计数**，临界值（45%–55%）必须回到月令与通根细辨。

### 5. 综合定档
- **身强（旺）**：得令 + 通根 + 党众，宜「克泄耗」。
- **身弱**：失令 + 无根 + 异党旺，宜「生扶」。
- **均衡**：力量相当，取调候或通关，或扶抑微调。
- **极端 → 从格/专旺**：日主极旺无制 → 专旺/从旺；日主极弱无根无生、满盘异党 → 从弱（从财/从杀/从儿/从势）。一旦成从，**顺其势者吉、逆其势者凶**，取用与常格相反。

## 二、取用神五法

用神 = 调节全局、补偏救弊、令命局趋于中和的那个五行/十神。**优先级：调候 ≥ 扶抑 > 通关 > 病药，从格则顺势。**

### 法一：扶抑（最常用）
- 身强：取**官杀（克）、食伤（泄）、财星（耗）**为用，抑其有余。
- 身弱：取**印星（生）、比劫（帮）**为用，补其不足。
- 口诀：强者宜泄宜克，弱者宜生宜帮。

### 法二：调候（寒暖燥湿，优先级常高于扶抑）
气候失衡时，调候用神优先。详见 `03_tiaohou_qiongtong.md`。
- 冬生（亥子丑月）水寒 → 急需**丙丁火**暖局，无火则寒凝不发。
- 夏生（巳午未月）火炎 → 急需**壬癸水**润局，无水则燥烈无成。
- 燥土（戌未）需水润，湿土（辰丑）需火暖。
- **调候不及，纵格局成立亦减分**；调候到位，是富贵的重要标志。

### 法三：通关
两种五行势均力敌、相战不下时，取**通关之神**化解。
- 如金木交战 → 取**水**通关（金生水、水生木）。
- 如水火相战 → 取**木**通关（水生木、木生火）。

### 法四：病药
命局中破坏中和的忌神为「病」，能去病的为「药」。
- 「有病方为贵，无伤不是奇；格中如去病，财禄两相宜。」
- 如官星为用却被伤官克破，伤官即「病」，能制伤官的印星即「药」。

### 法五：专旺 / 从格（顺势）
不能扶抑时顺其旺神。
- **专旺格**：曲直（木）、炎上（火）、稼穑（土）、从革（金）、润下（水），用神为旺神本身及其生助、泄秀之神，忌克制。
- **从格**：从财、从杀（官）、从儿（食伤）、从强、从势，用神为所从之神，最忌印比逆之。

## 三、流派仲裁：多个用神候选冲突时听谁的（决策树）

调候派（穷通宝鉴 / 03）、格局派（子平真诠 / 05 / 14）、旺衰中和派（滴天髓 / 13）、病药派（神峰通考 / 09）对同一命局常给出**不同的用神**。下面是把四派合一的固定裁决顺序，按优先级从高到低逐层过滤，**上一层定了就不被下一层推翻**，只在其内部细化。

### 裁决顺序（五道关，依次过）

**第一关 · 先验从格 / 专旺（最高否决权）**
日主极旺无制、或极弱无根无生且满盘异党 → 走从格 / 专旺，**顺其势取用，以下各派常规取用全部作废**。判据用旺衰五看 + `13` 滴天髓「旺极不可损、衰极不可益」。**条件极严，一点根气或有效生扶即不成从**，存疑则回常格按身弱论（见误区四）。不成从 → 进第二关。

**第二关 · 调候急症（气候否决权）**
查 `03` / `13` 寒暖燥湿。命局若犯调候急病（冬生水寒无丙丁、夏生火炎无壬癸、火炎土燥无水、金寒水冷无火），**调候用神优先级压过扶抑与格局**：先取调候之神。子平真诠亦自认「调候为急，权而用之」（`14` 五），金水伤官见官反吉即此理。
- 调候字**缺位且气候明显偏枯** → 调候之神即第一用神。
- 调候字**已在局中得力** → 调候已解，转第三关。

**第三关 · 旺衰扶抑定五行大方向（地基）**
调候不急或已解后，回日主旺衰定大方向，即滴天髓「旺则宜泄宜伤，衰则喜帮喜助」：
- 身强 → 用神方向在**克泄耗**（官杀 / 食伤 / 财）。
- 身弱 → 用神方向在**生扶**（印 / 比劫）。
扶抑只给「该用哪类五行」，不给精确落点 → 进第四关精定。

**第四关 · 格局月令把方向精确到十神（精修）**
在扶抑大方向之内，用 `05` / `14` 月令取格 + 顺逆 + 相神，把「五行方向」收敛到「具体哪个十神为用、哪个十神为相」：
- 例：身强需泄克，月令正官透干成官格 → 用神锁定**正官**（非泛泛「克泄耗」），相神取**财**（生官）或**印**（护官）。
- 格局派解决扶抑「只给方向、不给落点」的短板；相神是成格关键，**伤相甚于伤用**（`14` 三）。

**第五关 · 病药校验成败高低（扫尾）**
用神大致定后，用 `09` 病药说扫一遍：有没有忌神正克破用神 / 相神（病）？有没有制去它的字（药）？
- **有病有药** → 药为关键之神（多数即相神），格局成而见贵，「有病方为贵，无伤不是奇」。
- **有病无药** → 格减等，待岁运补药之年方发。
- **无病** → 中和清纯，用神直接到位。

### 一句话仲裁口诀

> **先验从格，再急调候，扶抑定向，格局定点，病药校验。** 上层未触发才轮到下层，上层定了下层只细化不推翻。

### 两派打架时的典型裁决

| 冲突场景 | 表面矛盾 | 裁决 | 依据 |
|---|---|---|---|
| **调候 vs 扶抑** | 身弱冬水命，扶抑要金水助身，调候要丙火暖局（火既泄又耗弱身） | 调候优先，先取火暖局救偏枯；但须权衡火不可过猛伤身，取「暖局又不重伤身」的平衡字（必要时火土金通盘配） | 调候关 > 扶抑关；`03`+`13` |
| **格局 vs 扶抑（同向）** | 月令七杀成杀格（格局要食制杀），日主身强（扶抑也要泄克） | 无真冲突：食制杀即是泄，两者合一，直接用食伤制杀 | 第三、四关同向叠加 |
| **格局 vs 扶抑（反向）** | 身弱却逢月令财格，格局顺用要食伤生财，身弱最忌再泄 | 扶抑优先：先取印比助身任财（财滋弱主），子平真诠亦云「财旺身弱喜印比」 | 扶抑关 > 格局关；`14` 财格 |
| **病药 vs 格局** | 病药找的「病」与格局派的「忌神」 | 多数同指一字（病=忌神，药=相神或制忌之神），措辞不同实质一致，并行不悖 | 第五关与第四关互证；`09` |

> **总则**：四派各管命局的不同层级，**层级分明**：调候管气候底盘、扶抑管旺衰方向、格局管十神落点、病药管成败校验。冲突几乎都能用「谁的层级更高」化解。真正两难（如调候与扶抑五行正相反且都急）时，回到滴天髓「中和」总目标，取那个**让全局最趋中和、副作用最小**的字，宁可用神力弱而平衡，不取用神力强而偏枯（`13` 二、三）。

## 四、用神 · 喜神 · 忌神 · 仇神 · 闲神

| 类别 | 定义 | 作用 |
|---|---|---|
| **用神** | 最能令命局中和的五行/十神 | 核心，定富贵层次 |
| **喜神** | 生助用神者 | 辅佐用神，多多益善 |
| **忌神** | 破坏用神、加剧失衡者 | 大运流年逢之为凶 |
| **仇神** | 生助忌神者 | 助纣为虐 |
| **闲神** | 于用忌皆无大碍者 | 中性，可被引动转吉凶 |

## 五、常见误区（断命前自检）

1. **未定旺衰就取用神**：根基错则全盘错，必先定旺衰。
2. **只数五行个数定强弱**：忽略月令司权、通根透干，是最大误判源。
3. **忽略调候**：冬无火、夏无水的命，扶抑取用也难发。
4. **见从就从**：从格条件极严，命局有一点根气或生扶即不成从，按身弱论。
5. **神煞当家**：神煞为辅，喧宾夺主则失准（见 `06_shensha.md`）。
